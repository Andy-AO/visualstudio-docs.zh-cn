# <a name="stepkind"></a>STEPKIND
指定步进的步骤类型。

## <a name="syntax"></a>语法

```cpp
enum enum_STEPKIND { 
   STEP_INTO      = 0,
   STEP_OVER      = 1,
   STEP_OUT       = 2,
   STEP_BACKWARDS = 3
};
typedef DWORD STEPKIND;
```

```csharp
public enum enum_STEPKIND { 
   STEP_INTO      = 0,
   STEP_OVER      = 1,
   STEP_OUT       = 2,
   STEP_BACKWARDS = 3
};
```

## <a name="fields"></a>字段
 `STEP_INTO`\
 步骤进入函数。

 `STEP_OVER`\
 在函数上执行步骤。

 `STEP_OUT`\
 退出函数。

 `STEP_BACKWARDS`\
 向后的步骤进入函数。

## <a name="remarks"></a>备注
 作为参数传递给[Step](../../../extensibility/debugger/reference/idebugprocess3-step.md)方法。
