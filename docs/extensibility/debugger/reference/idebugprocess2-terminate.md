# <a name="idebugprocess2terminate"></a>IDebugProcess2::Terminate
终止进程。

## <a name="syntax"></a>语法

```cpp
HRESULT Terminate( 
   void 
);
```

```csharp
int Terminate();
```

## <a name="return-value"></a>返回值
 如果成功，返回`S_OK`;否则，返回错误代码。

## <a name="remarks"></a>备注
 当进程终止时，该进程中的所有程序都终止;不允许再运行任何代码。
