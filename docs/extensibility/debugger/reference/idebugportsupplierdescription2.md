# <a name="idebugportsupplierdescription2"></a>IDebugPortSupplierDescription2
使[!INCLUDE[vsprvs](../../../code-quality/includes/vsprvs_md.md)]UI 来显示内的文本**传输信息**一部分**附加到进程**对话框。

```

```
