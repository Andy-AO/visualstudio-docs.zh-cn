# <a name="idebugmodule3isusercode"></a>IDebugModule3::IsUserCode
检索或不该模块是否表示用户代码的信息。

## <a name="syntax"></a>语法

```cpp
HRESULT IsUserCode(
   BOOL* pfUser
);
```

```csharp
int IsUserCode(
   out int pfUser
);
```

## <a name="parameters"></a>参数
`pfUser`\
[out]非零值 (`TRUE`) 模块所代表用户代码，如果零 (`FALSE`) 如果不是。

## <a name="return-value"></a>返回值
 如果成功，则返回`S_OK`; 否则为将返回错误代码。
