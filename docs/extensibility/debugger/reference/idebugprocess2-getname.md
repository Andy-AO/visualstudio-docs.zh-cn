# <a name="idebugprocess2getname"></a>IDebugProcess2::GetName
获取进程的标题、友好名称或文件名。

## <a name="syntax"></a>语法

```cpp
HRESULT GetName( 
   GETNAME_TYPE  gnType,
   BSTR*         pbstrName
);
```

```csharp
int GetName( 
   enum_GETNAME_TYPE  gnType,
   out string         pbstrName
);
```

## <a name="parameters"></a>参数
`gnType`\
[在][GETNAME_TYPE](../../../extensibility/debugger/reference/getname-type.md)枚举中指定要返回的名称类型的值。

`pbstrName`\
[出]返回进程的名称。

## <a name="return-value"></a>返回值
 如果成功，返回`S_OK`;否则，返回错误代码。
