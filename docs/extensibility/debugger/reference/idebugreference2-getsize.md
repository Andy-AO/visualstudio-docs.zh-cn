# <a name="idebugreference2getsize"></a>IDebugReference2::GetSize
获取大小，以字节为单位的引用的值。 留待将来使用。

## <a name="syntax"></a>语法

```cpp
HRESULT GetSize ( 
   DWORD* pdwSize
);
```

```csharp
int GetSize ( 
   out uint pdwSize
);
```

## <a name="parameters"></a>参数
 `pdwSize`\

 [out]返回的大小，以字节为单位的引用的值。

## <a name="return-value"></a>返回值
 始终返回 `E_NOTIMPL`。
