# <a name="pending_bp_state_flags"></a>PENDING_BP_STATE_FLAGS
指定挂起的断点状态标志。

## <a name="syntax"></a>语法

```cpp
enum enum_PENDING_BP_STATE_FLAGS { 
   PBPSF_NONE        = 0x0000,
   PBPSF_VIRTUALIZED = 0x0001
};
typedef DWORD PENDING_BP_STATE_FLAGS;
```

```csharp
public enum enum_PENDING_BP_STATE_FLAGS { 
   PBPSF_NONE        = 0x0000,
   PBPSF_VIRTUALIZED = 0x0001
};
```

## <a name="fields"></a>字段
 `PBPSF_NONE`占 位 符。

 `PBPSF_VIRTUALIZED`指定虚拟化挂起的断点，该断点将在每次加载新代码时绑定。

## <a name="remarks"></a>备注
 用于[PENDING_BP_STATE_INFO](../../../extensibility/debugger/reference/pending-bp-state-info.md)结构`flags`的成员。
