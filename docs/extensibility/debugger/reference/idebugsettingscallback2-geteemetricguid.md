# <a name="idebugsettingscallback2geteemetricguid"></a>IDebugSettingsCallback2::GetEEMetricGuid
检索在给定其名称的表达式计算器指标的唯一标识符。

## <a name="syntax"></a>语法

```cpp
HRESULT GetEEMetricGuid(
   REFGUID guidLang,
   REFGUID guidVendor,
   LPCWSTR pszMetric,
   GUID*   pguidValue
);
```

```csharp
HRESULT GetEEMetricGuid(
   ref Guid guidLang,
   ref Guid guidVendor,
   string   pszMetric,
   out Guid pguidValue
);
```

## <a name="parameters"></a>参数
`guidLang`\
[in]编程语言的唯一标识符。

`guidVendor`\
[in]供应商的唯一标识符。

`pszMetric`\
[in]指标的名称。

`pguidValue`\
[out]返回度量值的唯一标识符。

## <a name="return-value"></a>返回值
 如果成功，则返回`S_OK`; 否则为返回错误代码。
