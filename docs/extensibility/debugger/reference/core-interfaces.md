# <a name="core-interfaces"></a>核心接口
以下接口是用于通过使用扩展调试器的核心接口[!INCLUDE[vsipsdk](../../../extensibility/includes/vsipsdk_md.md)]。

## <a name="discussion"></a>讨论
 这些接口主要用于创建调试引擎 (DE)。 在此处按类别进行组织：

- [断点](#Breakpoints)

- [上下文](#Contexts)

- [Core Server](#CoreServer)

- [调试引擎](#DebugEngines)

- [文档](#Documents)

- [事件](#Events)

- [表达式](#Expressions)

- [内存](#Memory)

- [模块](#Modules)

- [端口](#Ports)

- [进程](#Processes)

- [程序](#Programs)

- [属性](#Properties)

- [堆栈帧](#StackFrames)

- [线程](#Threads)

- [类型可视化工具](#TypeVisualizers)

  可以实现接口的实体是：

- 调试引擎 (DE)

- 端口提供程序 (PS)

- 表达式计算器 (EE)

- Visual Studio (VS)
