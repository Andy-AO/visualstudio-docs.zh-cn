# <a name="idebugmessageevent2setresponse"></a>IDebugMessageEvent2::SetResponse
从消息框中设置响应（如果有）。

## <a name="syntax"></a>语法

```cpp
HRESULT SetResponse( 
   DWORD dwResponse
);
```

```csharp
int SetResponse( 
   uint dwResponse
);
```

## <a name="parameters"></a>参数
`dwResponse`\
[在]使用 Win32`MessageBox`函数的约定指定响应。 有关详细信息，请参阅[AfxMessageBox](/cpp/mfc/reference/cstring-formatting-and-message-box-display#afxmessagebox)功能。

## <a name="return-value"></a>返回值
 如果成功，返回`S_OK`;否则，返回错误代码。
