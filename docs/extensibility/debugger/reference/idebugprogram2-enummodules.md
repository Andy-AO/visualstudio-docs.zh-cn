# <a name="idebugprogram2enummodules"></a>IDebugProgram2::EnumModules
检索此程序已加载并正在执行的模块的列表。

## <a name="syntax"></a>语法

```cpp
HRESULT EnumModules( 
   IEnumDebugModules2** ppEnum
);
```

```csharp
int EnumModules( 
   out IEnumDebugModules2 ppEnum
);
```

## <a name="parameters"></a>参数
`ppEnum`\
[出]返回包含模块列表的[IEnumDebugModule2](../../../extensibility/debugger/reference/ienumdebugmodules2.md)对象。

## <a name="return-value"></a>返回值
 如果成功，返回`S_OK`;否则，返回错误代码。

## <a name="remarks"></a>备注
 模块是 DLL 或程序集，通常列在**模块**调试窗口中。
