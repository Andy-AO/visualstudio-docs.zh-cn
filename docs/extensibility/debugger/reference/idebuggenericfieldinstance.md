# <a name="idebuggenericfieldinstance"></a>IDebugGenericFieldInstance
表示托管代码泛型类型的字段的实例。

## <a name="syntax"></a>语法

```
IDebugGenericFieldInstance : IUnknown
```

## <a name="methods"></a>方法
 此接口实现以下方法：

|方法|描述|
|------------|-----------------|
|[GetTypeArguments](../../../extensibility/debugger/reference/idebuggenericfieldinstance-gettypearguments.md)|检索此实例的类型参数参数参数。|
|[TypeArgumentCount](../../../extensibility/debugger/reference/idebuggenericfieldinstance-typeargumentcount.md)|返回此实例的类型参数参数参数数。|
