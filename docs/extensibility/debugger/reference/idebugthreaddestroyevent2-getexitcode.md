# <a name="idebugthreaddestroyevent2getexitcode"></a>IDebugThreadDestroyEvent2::GetExitCode
获取线程的退出代码。

## <a name="syntax"></a>语法

```cpp
HRESULT GetExitCode ( 
   DWORD* pdwExit
);
```

```csharp
int GetExitCode ( 
   out uint pdwExit
);
```

## <a name="parameters"></a>参数
`pdwExit`\
[出]返回线程的退出代码。

## <a name="return-value"></a>返回值
 如果成功，返回`S_OK`;否则，返回错误代码。
