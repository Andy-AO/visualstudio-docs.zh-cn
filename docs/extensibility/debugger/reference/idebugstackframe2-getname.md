# <a name="idebugstackframe2getname"></a>IDebugStackFrame2::GetName
获取堆栈帧的名称。

## <a name="syntax"></a>语法

```cpp
HRESULT GetName ( 
   BSTR* pbstrName
);
```

```csharp
int GetName ( 
   out string pbstrName
);
```

## <a name="parameters"></a>参数
`pbstrName`\
[出]返回堆栈帧的名称。

## <a name="return-value"></a>返回值
 如果成功，返回`S_OK`;否则，返回错误代码。

## <a name="remarks"></a>备注
 堆栈帧的名称通常是要执行的方法的名称。
