# <a name="ieedatastoragegetsize"></a>IEEDataStorage::GetSize
返回此对象中包含的字节数。

## <a name="syntax"></a>语法

```cpp
HRESULT GetSize(
   ULONG* size
);
```

```csharp
int GetSize(
   out uint size
);
```

## <a name="parameters"></a>参数
`size`\
弄此对象中包含的字节数。

## <a name="return-value"></a>返回值
 如果成功， `S_OK` 则返回; 否则返回错误代码。

## <a name="remarks"></a>备注
 使用工作 [方式](../../../extensibility/debugger/reference/ieedatastorage-getdata.md) 方法检索实际数据字节。
