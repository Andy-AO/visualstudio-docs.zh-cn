# <a name="idebugcoreserver3queryislocal"></a>IDebugCoreServer3::QueryIsLocal
确定服务器是否为调用方的本地服务器。

## <a name="syntax"></a>语法

```cpp
HRESULT QueryIsLocal(
   void
);
```

```csharp
int QueryIsLocal();
```

## <a name="return-value"></a>返回值
 返回 `S_OK` ，指示服务器是本地的。 `S_FALSE`如果服务器是从 msvsmon.exe 的实例（通常用于远程调试）运行，则返回。
