# <a name="idebugprogramhost2gethostid"></a>IDebugProgramHost2::GetHostId
获取承载此程序的进程的进程标识符。

## <a name="syntax"></a>语法

```cpp
HRESULT GetHostId( 
   AD_PROCESS_ID* pdwId
);
```

```csharp
int GetHostId( 
   AD_PROCESS_ID[] pdwId
);
```

## <a name="parameters"></a>参数
`pdwId`\
[进出]使用进程标识符信息填充[AD_PROCESS_ID](../../../extensibility/debugger/reference/ad-process-id.md)结构。

## <a name="return-value"></a>返回值
 如果成功，返回`S_OK`;否则，返回错误代码。
