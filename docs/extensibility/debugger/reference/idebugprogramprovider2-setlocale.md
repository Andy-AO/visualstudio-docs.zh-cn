# <a name="idebugprogramprovider2setlocale"></a>IDebugProgramProvider2::SetLocale
建立用于任何特定于区域设置的资源的区域设置。

## <a name="syntax"></a>语法

```cpp
HRESULT SetLocale(
   WORD wLangID
);
```

```csharp
int SetLocale(
   ushort wLangID
);
```

## <a name="parameters"></a>参数
`wLangID`\
[在]要建立的语言 ID。 例如，英语为 1033。

## <a name="return-value"></a>返回值
 如果成功，返回`S_OK`;否则，返回错误代码。
