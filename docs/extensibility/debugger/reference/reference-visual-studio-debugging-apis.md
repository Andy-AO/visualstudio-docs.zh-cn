# <a name="reference-visual-studio-debugging-apis"></a>引用（Visual Studio 调试 API）

Visual Studio 调试 SDK 引用概述了调试器及其组件的运行时体系结构和功能的基本模型。 除了通过接口定义的行为外，本文档对它描述的组件和接口做出尽可能少的假设。

参考部分的目标是尽可能为各种语言、运行时间和操作系统提供支持。
