# <a name="idebugexpressionevaluatorsetlocale"></a>IDebugExpressionEvaluator::SetLocale
此方法设置用于创建可打印结果的语言。

## <a name="syntax"></a>语法

```cpp
HRESULT SetLocale( 
   WORD wLangID
);
```

```csharp
int SetLocale(
   ushort wLangID
);
```

## <a name="parameters"></a>参数
`wLangID`\
[在]语言标识符。

## <a name="return-value"></a>返回值
 如果成功，返回`S_OK`;否则，返回错误代码。

## <a name="remarks"></a>备注
 在加载表达式赋值器 （EE） 时，可以多次调用此方法，因此 EE 必须能够动态切换语言。 EE 使用此区域设置以适当的语言返回错误消息和字符串。
