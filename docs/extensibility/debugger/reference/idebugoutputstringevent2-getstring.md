# <a name="idebugoutputstringevent2getstring"></a>IDebugOutputStringEvent2::GetString
获取可显示消息。

## <a name="syntax"></a>语法

```cpp
HRESULT GetString( 
   BSTR* pbstrString
);
```

```csharp
int GetString( 
   out string pbstrString
);
```

## <a name="parameters"></a>参数
`pbstrString`\
弄返回可显示的消息。

## <a name="return-value"></a>返回值
 如果成功， `S_OK` 则返回; 否则返回错误代码。
