# <a name="idebugaliasgetname"></a>IDebugAlias::GetName
获取此别名的名称。

## <a name="syntax"></a>语法

```cpp
HRESULT GetName(
   BSTR* pbstrName
);
```

```csharp
int GetName(
   out string pbstrName
);
```

## <a name="parameters"></a>参数
`pbstrName`\
弄别名的名称。

## <a name="return-value"></a>返回值
 如果成功，将返回 S_OK;否则，将返回错误代码。
