# <a name="asyncvoidmethodbuilderm_objectidfordebugger-field"></a>异步VoidMethodBuilder.m_objectIdForDebugger字段
表示调试器用于唯一标识此生成器的懒惰初始化对象。

 **命名空间：**<xref:System.Runtime.CompilerServices?displayProperty=fullName>

 **程序集**：mscorlib（在 mscorlib.dll 中）

 由于您无法从 .NET 框架访问此内部成员，因此在通用中间语言 （CIL） 中提供了以下语法。

## <a name="syntax"></a>语法

```csharp
.field private object m_objectIdForDebugger
```
