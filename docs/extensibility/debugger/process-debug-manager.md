# <a name="process-debug-manager"></a>过程调试管理器
进程调试管理器 （PDM） 是 Visual Studio 的一个组件，用于管理程序和进程，使其可供会话调试管理器和调试引擎使用。

 PDM 管理所有可以调试的进程。 要调试程序，必须向 PDM 注册程序。 此注册在启动程序时通过端口或调试引擎完成。
