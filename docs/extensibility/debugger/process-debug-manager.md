# <a name="process-debug-manager"></a>进程调试管理器
进程调试管理器 (PDM) 是 Visual Studio 的一个组件，它可管理程序和进程，使其可用于会话调试管理器和调试引擎。

 PDM 管理可调试的所有进程。 若要进行调试，必须向 PDM 注册程序。 此注册是在启动程序时（通过端口或调试引擎）完成的。
