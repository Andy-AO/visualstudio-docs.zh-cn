# <a name="extend-properties-and-the-property-window"></a>扩展属性和属性窗口
属性**窗口**用于查看和编辑所选对象的设计时属性和事件。

 在 中选择 对象[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]时，该对象的公共属性将显示在 **"属性"** 窗口中。 有关详细信息，请参阅[属性窗口概述](../extensibility/internals/properties-window-overview.md)。
