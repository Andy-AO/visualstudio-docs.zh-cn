# <a name="web-projects"></a>Web 项目
Web 项目系统是创建 Web 项目的项目系统。 Web 项目反过来创建 Web 应用程序。 您可以使用传统的编程语言（如[!INCLUDE[vbprvb](../../code-quality/includes/vbprvb_md.md)]或[!INCLUDE[csprcs](../../data-tools/includes/csprcs_md.md)]）对网页进行编程，以便收集和处理用户的信息，将其存储在数据库中，等等。 具有关联代码的网页称为智能网页。
