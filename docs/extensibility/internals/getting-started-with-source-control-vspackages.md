# <a name="get-started-with-source-control-vspackages"></a>开始使用源代码管理 VSPackages

源代码管理 VSPackage 是源代码管理插件的更高级替代方法。 有关源代码管理插件的详细信息，请参阅[创建源代码管理插件](../../extensibility/internals/creating-a-source-control-plug-in.md)。 源代码管理 VSPackage 支持对源代码管理模型、功能和用户界面 （UI） 的完全控制，并将它作为 VSPackage[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]集成到环境中。
