# <a name="outlining-and-hidden-text-in-a-legacy-language-service"></a>旧版语言服务中的大纲显示和隐藏文本
大纲可以将一系列文本行折叠成一行。 例如，在C++所有方法都可以折叠到一行，仅显示方法签名。 隐藏文本是可以显示或隐藏的文本行的顺序。
