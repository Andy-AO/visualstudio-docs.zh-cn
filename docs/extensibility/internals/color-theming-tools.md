# <a name="color-theme-tools"></a>颜色主题工具
Visual Studio SDK 提供了多种工具，可帮助您创建和管理颜色主题。

- [VSIX 颜色编辑器](../../extensibility/internals/vsix-color-editor.md)工具可以为 Visual Studio 创建和编辑自定义颜色。

- [VSIX 颜色编译器](../../extensibility/internals/vsix-color-compiler.md)工具将现有的 Visual Studio 主题 *.xml*文件转换为 *.pkgdef*文件，用于扩展。
