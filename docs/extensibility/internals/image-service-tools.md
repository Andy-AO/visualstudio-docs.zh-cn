# <a name="image-service-tools"></a>图像服务工具
VS SDK 包括多个工具，可帮助扩展生成器使用 Visual Studio 图像服务。

- "[来自资源清单"](../../extensibility/internals/manifest-from-resources.md)工具获取图像资源列表（.png 或 .xaml 文件）并生成图像清单文件。

- "[清单到代码"](../../extensibility/internals/manifest-to-code.md)工具采用图像清单文件并生成包装文件，用于C++、C#、VB 或 .vsct 文件。

- [图像库查看器](../../extensibility/internals/image-library-viewer.md)可以加载、编辑和搜索图像清单。
