# <a name="monitoring-gpu-utilization"></a>监视 GPU 使用率

监视远程 Linux 计算机的 GPU 使用率：

1. 在“服务器资源管理器”中，展开“远程计算机”   。
2. 右键单击要监视的远程计算机  。

    ![GPU 热度地图](media/monitor-gpu/gpu-heatmap-0.png)

3. 单击“显示热度地图”  。

    ![GPU 热度地图](media/monitor-gpu/heatmap.png)
