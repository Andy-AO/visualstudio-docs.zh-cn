# <a name="c28020"></a>C28020
警告 C28020：在此调用中，表达式 \<expr > 不为 true

 当列出的 \_Satisfies \_ 表达式不为 true 时，将报告此警告。 这通常表示存在错误的参数。

 如果在函数声明时发生此警告，则表示批注指明了一种不可能的条件。
