# <a name="c6274"></a>C6274
警告 C6274：作为参数传递的非字符 \<number > 需要字符才能调用 \<function >

 此警告表明，格式字符串指定需要一个字符（例如 `%c` 或 `%C` 规范），但不会传递诸如 float、string 或 struct 这样的非整数。 此缺陷可能会导致错误的输出。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str);
}
```

 若要更正此警告，请使用以下代码：

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str[0]);
}
```

 下面的代码使用安全的字符串操作函数 `sprintf_s` 来更正此警告：

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf_s(buff,5,"%c", str[0]);
}
```
