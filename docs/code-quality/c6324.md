# <a name="c6324"></a>C6324
警告 C6324： \<function1 > 使用可能不正确：是否要使用 \<function2 >？

 此警告意味着使用了字符串比较函数应使用的字符串复制函数。 函数的使用不正确可能会导致意外的逻辑错误。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```
#include <string.h>

void f(char *title )
{
  if (strcpy (title, "Manager") == 0) // warning 6324
  {
    // code
  }
}
```

 若要更正此警告，请使用 `strcmp`，如以下代码所示：

```
#include <string.h>

void f(char *title )
{
  if (strcmp (title, "Manager") == 0)
  {
    // code
  }
}
```
