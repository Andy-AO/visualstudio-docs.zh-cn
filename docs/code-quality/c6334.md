# <a name="c6334"></a>C6334
警告 C6334： sizeof 运算符应用于带有运算符的表达式可能产生意外结果

 此警告表明 `sizeof` 运算符被滥用。 将 `sizeof` 运算符应用到表达式时，将生成所生成的表达式的类型的大小。

 例如，在以下代码中：

```
char     a[10];
size_t   x;

x = sizeof (a - 1);
```

 将为 `x` 分配值4（而不是9），因为所得表达式不再是指向数组的指针 `a`，而只是一个指针。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```
void f( )
{
  size_t x;
  char a[10];

  x= sizeof (a - 4);
  // code...
}
```

 若要更正此警告，请使用以下代码：

```
void f( )
{
  size_t x;
  char a[10];

  x= sizeof (a) - 4;
  // code...
}
```
