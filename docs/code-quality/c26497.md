# <a name="c26497-use_constexpr_for_function"></a>C26497 USE_CONSTEXPR_FOR_FUNCTION

如果需要编译时计算，则可以将此函数% function% 标记 `constexpr`。 请参阅[ C++核心准则 a-f](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rf-constexpr)。
