# <a name="c6385"></a>C6385
警告 C6385：数据无效：访问 \<buffer 名称 >，可读大小 \<size1 > 个字节，但 \<size2 > 字节数：行： x，y

 此警告表明指定缓冲区的可读范围可能小于用于读取它的索引。 尝试读取有效范围以外的数据会导致缓冲区溢出。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```
void f(int i)
{
   char a[20];
   char j;
   if (i <= 20)
   {
      j = a[i];
   }
}
```

 若要更正此警告，请使用以下代码：

```
void f(int i)
{
   char a[20];
   char j;
   if (i < 20)
   {
      j = a[i];
   }
}
```
