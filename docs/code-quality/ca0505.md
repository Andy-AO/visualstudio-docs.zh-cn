# <a name="ca0505"></a>CA0505

由于定义了 superceding 属性，将忽略不推荐使用的属性。

当不推荐使用和取代代码分析属性在项目或生成配置文件中都具有值时，将生成警告 CA505。 若要解决此问题，请删除不推荐使用的属性。
