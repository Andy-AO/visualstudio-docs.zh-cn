# <a name="c6294"></a>C6294
警告 C6294： for 循环定义错误：初始条件不满足测试。 循环正文未执行

 此警告表明无法执行 for 循环，因为终止条件为 true。 此警告表明无法正确捕获程序员的意图。

## <a name="example"></a>示例
 下面的示例代码生成此警告，因为 MAX_VALUE 为0：

```cpp
#define MAX_VALUE 0
void f()
{
  int i;
  for (i = 0; i < MAX_VALUE; i++)
  {
    // code
  }
}
```

 下面的示例代码通过将 MAX_VALUE 的值更改为25来更正此警告。

```cpp
#define MAX_VALUE 25
void f()
{
  int i;
  for (i = 0; i < MAX_VALUE; i++)
  {
    // code
  }
}
```
