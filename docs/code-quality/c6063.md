# <a name="c6063"></a>C6063
警告 C6063： > 与转换说明符对应的 \<function 缺少字符串参数 \<number >

 此警告表示提供的参数不足以匹配格式字符串;至少缺少一个参数是字符串。 此缺陷可能会导致崩溃和缓冲区溢出（如果被调用的函数为 `sprintf` 系列）以及可能的错误输出。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```
#include <string.h>
void f( )
{
  char buff[15];
  sprintf(buff, "%s %s", "Hello, World!");
}
```

 若要更正此警告，请提供其他参数，如以下代码所示：

```
#include <string.h>
void f( )
{
  char buff[15];
  sprintf(buff, "%s %s ", "Hello","World");
}
```

 以下代码使用安全字符串操作函数更正此警告：

```
#include <string.h>
void f( )
{
  char buff[15];
  sprintf_s( buff, sizeof(buff),"%s", "Hello, World!" );
}
```
