# <a name="c28232"></a>C28232
警告 C28232：\_Pre @ no__t-1、\_Post @ no__t 或 \_Deref @ no__t-5 未应用于任何批注

 此警告意味着 `_Pre_`、`_Post_` 或 `_Deref_` 运算符出现在没有后续函数批注的批注表达式中；修饰符将被忽略，但这表示存在错误编码的批注。
