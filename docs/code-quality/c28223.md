# <a name="c28223"></a>C28223
警告 C28223：未找到批注和参数所需的标记/标识符

 此警告意味着批注的参数应为标识符，并且遇到其他某个符号。 这通常表示存在错误编码的批注宏。 在此位置使用 C C++或关键字会导致此错误。
