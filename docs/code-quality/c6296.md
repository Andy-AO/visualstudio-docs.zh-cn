# <a name="c6296"></a>C6296
警告 C6296： for 循环定义错误：循环体仅执行一次

 此警告意味着 for 循环可能不按预期方式工作。 如果索引未签名，并且循环从零开始计数，则其主体只运行一次。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```
void f( )
{
   unsigned int i;

   for (i = 0; i < 100; i--)
   {
      // code ...
   }
}
```

 若要更正此警告，请使用以下代码：

```
void f( )
{
   unsigned int i;

   for (i = 0; i < 100; i++)
   {
      // code ...
   }
}
```
