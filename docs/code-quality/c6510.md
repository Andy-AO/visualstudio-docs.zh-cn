# <a name="c6510"></a>C6510
警告 C6510：无效的批注： "NullTerminated" 属性只能在其元素为整型或指针类型的缓冲区上使用：函数 "\<函数 >" \<参数 >。

 此警告表示**NullTerminated**属性的使用不正确（以 "`_z`" 结尾的属性）。 只能对指针或数组类型使用此属性类型。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```cpp
#include <sal.h>

void f(_In_z_ char x)
{
    // code ...
}
```

 若要更正此警告，请使用以下代码：

```cpp
#include <sal.h>

void f(_In_z_ char * x)
{
    // code ...
}
```
