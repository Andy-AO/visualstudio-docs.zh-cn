# <a name="c6336"></a>C6336
警告 C6336：算术运算符的优先级高于问号运算符，请使用括号来阐明目的

 此警告表明可能存在运算符优先级问题。 "+"、"-"、"*" 和 "/" 运算符的优先级高于 "？" 运算符。 如果表达式中的优先级不正确，请使用括号更改运算符优先级。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```cpp
int Count();

void f(int flag)
{
  int result;
  result = Count() + flag ? 1 : 2;
  // code...
}
```

 若要更正此警告，请添加括号，如以下代码所示：

```cpp
int Count();

void f(int flag)
{
  int result;
  result = Count() + (flag ? 1 : 2);
  // code...
}
```
