# <a name="c6250"></a>C6250
警告 C6250:调用\<函数 > VirtualFree 不使用 MEM_RELEASE 标志可能会释放内存，但不是地址说明符 (Vad); 结果导致地址空间泄漏

 此警告意味着调用`VirtualFree`而无需`MEM_RELEASE`标志仅解除页，并不会释放它们。 若要解除并释放页，使用`MEM_RELEASE`调用中的标志`VirtualFree`。 如果区域中的所有页都都已提交，该函数首先解除，然后释放它们。 此操作后，在页处于可用状态。 如果指定此标志`dwSize`必须为零，并`lpAddress`必须指向返回的基址`VirtualAlloc`函数时该区域已保留。 如果不满足这些条件之一，则函数将失败。

 如果你的代码更高版本通过调用释放的地址空间，则可以忽略此警告`VirtualFree`与`MEM_RELEASE`标志。

 有关详细信息请参阅[VirtualAlloc](http://go.microsoft.com/fwlink/?LinkId=181201)并[VirtualFree](http://go.microsoft.com/fwlink/?LinkId=181202)。

 使用 VirtualAlloc 和 VirtualFree 有内存泄漏和异常方面的许多缺陷。 若要避免这些类型的泄漏和异常问题完全，使用的机制，通过提供C++标准模板库 (STL)。 其中包括[shared_ptr](/cpp/standard-library/shared-ptr-class)， [unique_ptr](/cpp/standard-library/unique-ptr-class)，并[向量](/cpp/standard-library/vector)。 有关详细信息，请参阅[智能指针](/cpp/cpp/smart-pointers-modern-cpp)并[C++标准库](/cpp/standard-library/cpp-standard-library-reference)。

## <a name="example"></a>示例
 下面的示例代码将生成此警告：

```
#include <windows.h>
#include <stdio.h>
#define PAGELIMIT 80

DWORD dwPages = 0;  // count of pages
DWORD dwPageSize;   // page size

VOID f( )
{
  LPVOID lpvBase;            // base address of the test memory
  SYSTEM_INFO sSysInfo;      // system information

  GetSystemInfo(&sSysInfo);
  dwPageSize = sSysInfo.dwPageSize;

  // Reserve pages in the process's virtual address space
  lpvBase = VirtualAlloc(
                         NULL,                // system selects address
                         PAGELIMIT*dwPageSize, // size of allocation
                         MEM_RESERVE,
                         PAGE_NOACCESS);
  //
  // code to access memory
  // ...

  if (lpvBase != NULL)
  {
    if (VirtualFree( lpvBase, 0, MEM_DECOMMIT )) // decommit pages
      {
        puts ("MEM_DECOMMIT Succeeded");
      }
      else
      {
       puts("MEM_DECOMMIT failed");
      }
  }
  else
  {
    puts("lpvBase == NULL");
  }
}
```

```
#include <windows.h>
#include <stdio.h>
#define PAGELIMIT 80

DWORD dwPages = 0;  // count of pages
DWORD dwPageSize;   // page size

VOID f( )
{
  LPVOID lpvBase;            // base address of the test memory
  SYSTEM_INFO sSysInfo;      // system information

  GetSystemInfo(&sSysInfo);
  dwPageSize = sSysInfo.dwPageSize;

  // Reserve pages in the process's virtual address space
  lpvBase = VirtualAlloc(
                         NULL,                // system selects address
                         PAGELIMIT*dwPageSize, // size of allocation
                         MEM_RESERVE,
                         PAGE_NOACCESS);
  //
  // code to access memory
  // ...

  if (lpvBase != NULL)
  {
   if (VirtualFree(lpvBase, 0,MEM_RELEASE )) // decommit & release
   {
     // code ...
   }
   else
   {
     puts("MEM_RELEASE failed");
   }
  }
  else
  {
    puts("lpvBase == Null ");
    // code...
  }
}
```
