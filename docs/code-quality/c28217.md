# <a name="c28217"></a>C28217
警告 C28217：对于函数，批注的参数数目与在文件中找到的数目不匹配

 当前行和消息行中的批注不一致。 如果对批注使用标准宏，则这种情况应该是不可能的；正常使用时不应出现此警告。
