# <a name="c26463-use_const_for_elements"></a>C26463 USE_CONST_FOR_ELEMENTS
  数组 "% array%" 的元素仅分配一次，将元素标记 `const`。 请参阅[ C++核心准则。 4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction)。
