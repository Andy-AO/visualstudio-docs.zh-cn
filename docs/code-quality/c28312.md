# <a name="c28312"></a>C28312
警告 28312:重复声明上的 < 批注名称 > 批注\<函数 >\<参数 > 是过时的 SAL 版本。

 批注是旧版本，应更新为等效[SAL2](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md)。 如果前一个不一致批注警告已发出，则此警告不会发出，此警告对于每个声明仅报告一次。 有关更多过时的 SAL 的信息，请查看声明的其他部分。
