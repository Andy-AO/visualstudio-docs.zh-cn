# <a name="c28218"></a>C28218
警告 C28218：对于函数参数，批注的参数与在文件中找到的参数不匹配

 当前行和消息行中的批注不一致。 如果对批注使用标准宏，则这种情况应该是不可能的；正常使用时不应出现此警告。
