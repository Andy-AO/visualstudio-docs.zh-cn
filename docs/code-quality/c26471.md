# <a name="c26471-no_reinterpret_cast_from_void_ptr"></a>C26471 NO_REINTERPRET_CAST_FROM_VOID_PTR

不要使用 `reinterpret_cast`。 Void * 的强制转换可以使用 `static_cast`。 请参阅[ C++核心准则类型。 1](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-reinterpretcast)。
