# <a name="c26138"></a>C26138
警告 C26138:持有锁时暂停协同程序\<锁 >。

协同程序挂起时持有锁时，会发出警告警告 C26138。 一般情况下，我们无法知道多长时间将协同程序保留在挂起状态，因此此模式可能会导致再临界区比预期。

## <a name="example"></a>示例
 以下代码将生成 C26138。

```cpp

#include <experimental/generator>
#include <future>
#include <mutex>

using namespace std::experimental;

std::mutex global_m;
_Guarded_by_(global_m) int var = 0;

generator<int> mutex_acquiring_generator() {
  global_m.lock();
  ++var;
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  global_m.unlock();
}

generator<int> mutex_acquiring_generator_report_once() {
  global_m.lock();
  ++var;
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  global_m.unlock();
}
```

## <a name="example"></a>示例
 下面的代码将会更正这些警告。

```cpp

#include <experimental/generator>
#include <future>
#include <mutex>

using namespace std::experimental;

std::mutex global_m;
_Guarded_by_(global_m) int var = 0;

generator<int> mutex_acquiring_generator2() {
  {
    global_m.lock();
    ++var;
    global_m.unlock();
  }
  co_yield 1;                           // no 26138, global_m is already released above.
}
```
