# <a name="c28204"></a>C28204
警告 C28204： \<function >：只有此重载中的一个重载，\<filename > （\<line >）中的一个重载用于 \<paramname > 的批注：要么都不能对其进行批注。

 当批注中存在错误时将报告此警告。
