# <a name="c28208"></a>C28208
警告 C28208： > 函数 \<function 以前使用不同的参数列表在 \<file > （\<line >）定义。 一些分析工具将产生不正确的结果

 当函数的已知定义与另一个匹配项不匹配时，将报告此警告。
