# <a name="c28208"></a>C28208
警告 C28208:函数\<函数 > 以前定义过在不同的参数列表\<文件 > (\<行 >)。 一些分析工具将产生不正确的结果

 函数的已知的定义与另一个匹配项不匹配时，将报告此警告。
