# <a name="c6509"></a>C6509
警告 C6509：无效的批注：无法从前置条件中引用 "return"

 此警告意味着在前置条件中不能使用 @no__t 0 关键字。 `return` 关键字用于终止函数的执行并将控制返回给调用函数。

## <a name="example"></a>示例
 下面的代码将生成此警告，因为在前置条件中使用了 `return`：

```cpp
#include <sal.h>

int f (_In_reads_(return) char *pc)
{
    // code ...
    return 1;
}
```

 若要更正此警告，请使用以下代码：

```cpp
#include <sal.h>

int f (_In_reads_(i) char *pc, int i)
{
    // code ...
    return 1;
}
```
