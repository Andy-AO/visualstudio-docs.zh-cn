# <a name="c6306"></a>C6306
警告 C6306：不正确地调用 \<函数 >：请考虑使用接受 va_list 作为参数的 \<函数 >

 此警告表明函数调用不正确。 `printf` 系列包含多个函数，这些函数采用参数的可变列表;但是，不能使用 `va_list` 参数调用这些函数。 有一个对应的 `vprintf` 系列函数可用于此类调用。 调用错误的打印功能将导致输出错误。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```cpp
#include <stdio.h>
#include <stdarg.h>

void f(int i, ...)
{
   va_list v;

   va_start(v, i);
   //code...
   printf("%s", v); // warning 6306
   va_end(v);
}
```

 若要更正此警告，请使用以下代码：

```cpp
#include <stdio.h>
#include <stdarg.h>

void f(int i, ...)
{
   va_list v;

   va_start(v, i);
   //code...
   vprintf_s("%d",v);
   va_end(v);
}
```
