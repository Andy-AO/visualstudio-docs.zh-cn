# <a name="c28219"></a>C28219
警告 C28219：批注中的批注参数所需的枚举成员

 批注的参数应为命名 `enum` 类型的成员，但遇到了其他某个符号;使用该 @no__t 类型的成员。 这通常表示存在错误编码的批注宏。
