# <a name="c26800"></a>C26800
警告 C26800:使用移动从对象：\<锁 >。

从移动之后使用变量时，会触发警告 C26800。 变量被视为从移动后它传递给一个函数作为右值引用。 有用于分配、 析构和重置功能，如 std::vector::clear 某些状态等一些合法例外情况。

## <a name="example"></a>示例
 以下代码将生成 C26800。

```cpp

#include <utility>

struct X {
    X();
    X(const X&);
    X(X&&);
    X &operator=(X&);
    X &operator=(X&&);
    ~X();
};

template<typename T>
void use_cref(const T&);

void test() {
    X x1;
    X x2 = std::move(x1);
    use_cref(x1);                // @expected(26800)
}
```

## <a name="example"></a>示例
 下面的代码不会生成 C26800。

```cpp

#include <utility>

struct MoveOnly {
    MoveOnly();
    MoveOnly(MoveOnly&) = delete;
    MoveOnly(MoveOnly&&);
    MoveOnly &operator=(MoveOnly&) = delete;
    MoveOnly &operator=(MoveOnly&&);
    ~MoveOnly();
};

template<typename T>
void use(T);

void test() {
    MoveOnly x;
    use(std::move(x)); // no 26800
}
```
