# <a name="c6315"></a>C6315
警告 C6315：运算顺序不正确：按位与的优先级高于按位或。 添加括号来阐明目的

 此警告意味着测试上下文中的表达式同时包含按位 "与" （`&`）和按位 "或" （`|`）运算，但由于上一次执行了按位 "或" 运算而导致常量。 应添加括号来阐明目的。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```
void f( int i )
{
  if ( i & 2 | 4 ) // warning
  {
    // code
  }
}
```

 若要更正此警告，请添加括号，如以下代码所示：

```
void f( int i )
{
  if ( i & ( 2 | 4 ) )
  {
    // code
  }
}
```
