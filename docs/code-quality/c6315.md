# <a name="c6315"></a>C6315
警告 C6315:运算顺序不正确： 按位的和的优先级高于按位的或。 添加括号明确指定优先级

 此警告意味着测试上下文中的表达式同时包含按位的和 (`&`) 和按位-或 (`|`) 操作，但会导致一个常量，因为的按位的或上一次操作。 应添加括号明确指定优先级。

## <a name="example"></a>示例
 下面的代码生成此警告：

```
void f( int i )
{
  if ( i & 2 | 4 ) // warning
  {
    // code
  }
}
```

 若要更正此警告，请添加括号，如下面的代码中所示：

```
void f( int i )
{
  if ( i & ( 2 | 4 ) )
  {
    // code
  }
}
```
