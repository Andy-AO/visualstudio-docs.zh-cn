# <a name="c28137"></a>C28137
警告 C28137：变量参数应为（文本）常量

 当函数调用缺少必需的（文本）常量时，会报告此警告。 请参阅函数的文档。

## <a name="example"></a>示例
 例如， [ExAcquireResourceExclusiveLite](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-exacquireresourceexclusivelite)例程需要 `Wait` 参数的值 TRUE 或 FALSE。 下面的示例代理将生成此警告：

```
ExAcquireResourceExclusiveLite(Resource, Wait);
```

 下面的代码示例可避免此警告：

```
ExAcquireResourceExclusiveLite(Resource, TRUE);
```
