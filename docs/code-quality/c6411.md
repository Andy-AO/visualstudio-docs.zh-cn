# <a name="c6411"></a>C6411
警告 C6411:可能从缓冲区读取的数据无效。

 此警告指示的索引，用于从缓冲区读取的值可以超过缓冲区的可读的大小。 当无法缩小表示缓冲区大小的复杂表达式或用于缓冲区的索引值时，代码分析工具也会报告此警告，因此可能会错误地报告此警告。

## <a name="example"></a>示例
 下面的代码将生成此警告。

```
char *a = new char[strlen(InputParam)];
delete[] a;
a[10];
```

 下面的代码将更正此错误。

```
int i = strlen(InputParam);
char *a = new char[i];
if (i > 10) a[10];
delete[] a;
```
