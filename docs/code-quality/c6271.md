# <a name="c6271"></a>C6271
警告 C6271：传递到 \<function >：参数 \<number > 的额外参数未由格式字符串使用

 此警告表示提供的其他参数超出了格式字符串指定的参数。 此缺陷本身将不会有任何明显的影响，尽管这表明程序员的意图并未反映在代码中。

## <a name="example"></a>示例
 下面的示例代码生成此警告：

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf(buff,"%d",1,2);
}
```

 若要更正此警告，请使用下面的示例代码：

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf(buff,"%d, %d",1,2);
}
```

 下面的示例代码调用安全字符串操作函数 `sprintf_s` 来更正此警告：

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];

   sprintf_s( buff, 5,"%s %d", 1,2 ); //safe version
}
```
