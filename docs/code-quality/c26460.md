# <a name="c26460-use_const_reference_arguments"></a>C26460 USE_CONST_REFERENCE_ARGUMENTS

  函数 "% function%" 的引用参数 "% argument%" 可以标记为 `const`。 请参阅[ C++核心准则 con。 3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref)。
