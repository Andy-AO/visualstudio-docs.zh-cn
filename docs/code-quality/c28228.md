# <a name="c28228"></a>C28228
警告 C28228：批注参数：不能在批注中使用类型

 此警告意味着参数的类型不受支持。 批注只能使用一组有限的类型作为参数。 如果正对批注使用标准宏，则这应不是可能实现的；此警告在正常使用时应不显示。
