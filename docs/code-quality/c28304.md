# <a name="c28304"></a>C28304
警告 C28304：对于C++引用参数 < parameter_name >，发现将错误放置的 `_Notref_` 运算符应用于 \<标记 >。

 `_Notref_` 运算符应只在涉及 C++ 引用参数的特殊情况下使用，并且只能位于系统提供的宏中。 运算符后必须紧接 `_Deref_` 运算符或功能性批注。
