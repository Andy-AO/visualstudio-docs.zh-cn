# <a name="c6295"></a>C6295
警告 C6295： for 循环定义错误： \<变量 > 值的范围为 "min" 到 "max"。 无限期执行

 此警告意味着 for 循环可能不按预期方式工作。 For 循环使用 > = 为零（0）测试无符号值。 结果始终为 true，因此循环是无限的。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```cpp
void f( )
{
  for (unsigned int i = 100; i >= 0; i--)
  {
    // code ...
  }
}
```

 若要更正此警告，请使用以下代码：

```cpp
void f( )
{
   for (unsigned int i = 100; i > 0; i--)
   {
      // code ...
   }
}
```
