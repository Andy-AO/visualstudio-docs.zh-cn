# <a name="ca0504"></a>CA0504
已忽略规则目录, 因为它不存在

当找不到项目或生成配置文件中的**CodeAnalysisRuleDirectories**属性中指定的目录时, 将生成警告 CA504。 若要解决此问题, 请输入有效的目录路径。
