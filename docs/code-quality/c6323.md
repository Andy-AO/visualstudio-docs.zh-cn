# <a name="c6323"></a>C6323
警告 C6323-对布尔类型使用算术运算符

 如果对布尔数据类型使用算术运算符，则会出现此警告。 使用错误的运算符可能产生不正确的结果。 它还指示程序员的意图不反映在代码中。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```cpp
int test(bool a, bool b)
{
    int c = a + b;     //C6323
    return c;
}
```

 若要更正此警告，请使用正确的数据类型和运算符。

```cpp
int test(int a, int b)
{
    int c = a + b;
    return c;
}
```
