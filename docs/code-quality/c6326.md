# <a name="c6326"></a>C6326
警告 C6326：常数与另一个常数的可能比较

 此警告表示存在一个常数与另一个常数（即冗余代码）的可能比较。 必须进行检查以确保在代码中正确捕获意向。 在某些情况下，可以简化测试条件以获得相同的结果。

## <a name="example"></a>示例
 下面的代码将生成此警告，因为比较了两个常量：

```
#define LEVEL
const int STD_LEVEL = 5;

const int value =
#ifdef LEVEL
10;
#else
5;
#endif

void f()
{
  if( value > STD_LEVEL)
  {
    // code...
  }
  else
  {
    // code...
  }
}
```

 下面的代码演示了通过使用 #ifdef 语句确定应执行的代码来更正此警告的一种方法：

```
#define LEVEL
const int STD_LEVEL = 5;

const int value =
#ifdef LEVEL
10;
#else
5;
#endif

void f ()
{
#ifdef LEVEL
  {
    // code...
  }
#else
  {
    // code...
  }
#endif
}
```
