# <a name="c28282"></a>C28282
警告 C28282：格式字符串必须位于前置条件中

 此警告意味着发现 `__drv_formatString` 批注，此批注未位于 `_Pre_` (`__drv_in`) 中（函数参数默认为 `_Pre_`）。 检查此批注是否用于带 `_Post_` (`__drv_out`) 批注的显式块。 如果是，则从将其置于 `_Post_` 上下文的任何封闭块中移除此批注。
