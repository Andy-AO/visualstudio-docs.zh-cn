# <a name="c6246"></a>C6246
警告 C6246：@No__t-0variable > 的局部声明隐藏了外部范围内同名的声明。 其他信息：请参阅 \<location > 上的上一个声明。

 此警告意味着在本地范围内两个声明具有相同的名称。 外部范围的名称被内部范围内的声明隐藏。 外部范围声明的任何预期用途都将导致使用局部声明。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```
#include <stdlib.h>
#define UPPER_LIMIT 256
int DoSomething( );

int f( )
{
  int i = DoSomething( );
  if (i > UPPER_LIMIT)
  {
    int i;
    i = rand( );
  }
  return i;
}
```

 若要更正此警告，请使用另一个变量名称，如下面的代码所示：

```
#include <stdlib.h>
#define UPPER_LIMIT 256
int DoSomething( );

int f ( )
{
  int i = DoSomething( );
  if (i > UPPER_LIMIT)
  {
    int j = rand( );
    return j;
  }
  else
  {
    return i;
  }
}
```

 此警告仅标识范围重叠。
