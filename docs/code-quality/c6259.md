# <a name="c6259"></a>C6259
警告 C6259：无法访问带标签的代码：（> \<expression 中的 & \<constant >）的计算结果不能为 \<case >

 此警告表明 switch 表达式中按位与（@no__t 0）比较的结果导致无法访问的代码。 仅可访问与 switch 表达式中的常量匹配的 case 语句;所有其他 case 语句均不可访问。

## <a name="example"></a>示例
 下面的示例代码将生成此警告，因为表达式 `switch``(rand() & 3)` 的计算结果不能为 case 标签（@no__t `4`）：

```
#include <stdlib.h>

void f()
{
     switch (rand () & 3) {
        case 3:
            /* Reachable */
            break;
        case 4:
            /* Not reachable */
            break;
        default:
            break;
    }
}
```

 若要更正此警告，请删除无法访问的代码或确认 case 语句中使用的常量是正确的。 以下代码将删除无法访问的 case 语句：

```
#include <stdlib.h>

void f()
{
     switch (rand () & 3) {
        case 3:
            /* Reachable */
            break;
        default:
            break;
    }
}
```
