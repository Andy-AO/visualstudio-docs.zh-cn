# <a name="c28194"></a>C28194
警告 C28194：该函数被声明为对变量中的值进行了别名，并已退出而没有这样做

 此警告意味着所分析函数的函数原型有一个 `__drv_isAliased` 批注，这表示它将为指定的自变量（即，以*它将在*从函数返回后保留的方式分配该值）。 但是，此函数不会沿批注所指示的路径用别名来命名此自变量。 大多数用别名来命名某个变量的函数都会将它的值保存到全局数据结构。
