# <a name="c6272"></a>C6272
警告 C6272：作为参数传递的非浮点型 \<number > 在调用时需要 float \<function >

 此警告表明，格式字符串指定需要 float，例如，`printf,` 的 @no__t 0 或 @no__t 规范，但要传递的是一个非浮点型（如整数或字符串）。 此缺陷可能会导致不正确的输出;但是，在某些情况下，可能会导致崩溃。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %f","a",i);
}
```

 若要更正此警告，请使用 `%i` 而不是 `%f` 规范，如以下代码所示：

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %i","a",i);
}
```

 下面的代码使用 `sprintf_s` 的安全字符串操作函数来更正此警告：

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf_s(buff,5,"%s %i","a",i); // safe version
}
```
