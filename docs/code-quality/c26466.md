# <a name="c26466-no_static_downcast_polymorphic"></a>C26466 NO_STATIC_DOWNCAST_POLYMORPHIC
  不要使用 `static_cast` type.2。 从多态类型强制转换应使用 dynamic_cast。 请参阅[ C++核心准则类型。 2](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-downcast)。
