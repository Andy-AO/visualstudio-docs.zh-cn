# <a name="c6287"></a>C6287
警告 C6287：冗余代码: 左右两侧的子表达式完全相同

 此警告表示在表达式中检测到冗余元素。

 在不检查代码的情况下，很难判断此问题的严重性。 自身的重复测试是无害的，但是删除第二个测试的后果可能很严重。 应检查代码以确保未省略测试。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```cpp
void f(int x)
{
  if ((x == 1) && (x == 1))
  {
    //logic
  }
  if ((x != 1) || (x != 1))
  {
    //logic
  }
}
```

 下面的代码演示了更正此警告的各种方式：

```cpp
void f(int x, int y)
{
  /* Remove the redundant sub-expression: */
  if (x == 1)
  {
     // logic
  }
  if (x != 1)
  {
    // logic
  }
  /* or test the missing variable: */
  if ((x == 1) && (y == 1))
  {
     // logic
  }
  if ((x != 1) || (y != 1))
  {
     // logic
  }
}
```
