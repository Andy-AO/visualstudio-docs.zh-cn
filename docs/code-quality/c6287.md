# <a name="c6287"></a>C6287
警告 C6287： 冗余代码： 左、 右子表达式完全相同

 此警告指示在表达式中检测到冗余元素。

 很难判断此问题的严重性而无需检查代码。 在其自身上的重复测试是无害的但删除第二个测试的后果可能非常严重。 应检查代码，以确保测试不被省略。

## <a name="example"></a>示例
 下面的代码生成此警告：

```
void f(int x)
{
  if ((x == 1) && (x == 1))
  {
    //logic
  }
  if ((x != 1) || (x != 1))
  {
    //logic
  }
}
```

 下面的代码显示了各种方法来更正此警告：

```
void f(int x, int y)
{
  /* Remove the redundant sub-expression: */
  if (x == 1)
  {
     // logic
  }
  if (x != 1)
  {
    // logic
  }
  /* or test the missing variable: */
  if ((x == 1) && (y == 1))
  {
     // logic
  }
  if ((x != 1) || (y != 1))
  {
     // logic
  }
}
```
