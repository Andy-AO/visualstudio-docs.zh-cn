# <a name="c6305"></a>C6305
警告 C6305: sizeof 和 countof 量值之间可能不匹配

 此警告意味着变量持有锁`sizeof`正在结果加上或从指针减去或`countof`表达式。 这会导致意外缩小指针算法。

## <a name="example"></a>示例
 下面的代码生成此警告：

```
void f(int *p)
{
  int cb=sizeof(int);
  //code...
  p +=cb; // warning 6305
}
```

 若要更正此警告，请使用以下代码：

```
void f(int *p)
{
  // code...
  p += 1;
}
```
