# <a name="ca0053"></a>CA0053
无法加载规则程序集。

这表示反射无法加载规则程序集。 如果用户具有未正确生成的自定义规则，或者分析工具由于其他原因（如文件系统访问被拒绝）无法加载程序集，则可能会发生这种情况。
