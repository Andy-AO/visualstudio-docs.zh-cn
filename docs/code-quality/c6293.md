# <a name="c6293"></a>C6293
警告 C6293： for 循环定义错误：从最小值向下计数

 此警告意味着 for 循环可能不按预期方式工作。 当循环从最小值向下计数但具有更高的终止条件时，会发生这种情况。

 带有负数增量的带符号或无符号索引变量将导致循环计数为负，直到发生溢出。 这将终止循环。

## <a name="example"></a>示例
 下面的示例代码生成此警告：

```cpp
void f( )
{
   signed char i;

   for (i = 0; i < 100; i--)
   {
      // code ...
   }
}
```

 若要更正此警告，请使用以下代码：

```cpp
void f( )
{
   signed char i;

   for (i = 0; i < 100; i++)
   {
      // code ...
   }
}
```
