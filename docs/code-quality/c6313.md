# <a name="c6313"></a>C6313
警告 C6313:运算符不正确：无法通过按位测试零值标志的和。 使用相等测试来检查零值标志

 此警告指示作为参数的按位提供零的常量值的和 (`&`) 在测试上下文中的运算符。 所生成的表达式是常量，并计算结果为 false;结果是不同于预期。

 这通常由于使用按位-和要测试其值为零的标志。 若要测试零值标志，为确定相等性测试必须执行，例如，使用`==`或`!=`。

## <a name="example"></a>示例
 下面的代码生成此警告：

```
#define FLAG 0

void f(int Flags )
{
  if (Flags & FLAG)
  {
    // code
  }
}
```

 若要更正此警告，请使用以下代码：

```
#define FLAG 0

void f(int Flags )
{
  if (Flags == FLAG)
  {
    // code
  }
}
```
