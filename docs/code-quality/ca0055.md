# <a name="ca0055"></a>CA0055
无法加载文件:\<路径 >

代码分析工具无法加载为分析指定的目标文件。 如果找不到文件或拒绝访问文件, 则会发生这种情况。 通常, 如果用户选择在 Visual Studio 中重写输入程序集属性, 或手动调用 fxcopcmd, 则会发生这种情况。
