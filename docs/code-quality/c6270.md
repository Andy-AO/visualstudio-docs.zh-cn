# <a name="c6270"></a>C6270
警告 C6270：缺少 \<function > 的 float 参数：添加对应于转换说明符的 float 参数 \<number >

 此警告表示提供的参数不足以匹配格式字符串;缺少的参数中至少有一个是浮点数。 除了可能的错误输出，此缺陷可能会导致故障。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```
#include <stdio.h>
#include <string.h>

void f()
{
  char buff [25];
  sprintf(buff,"%s %f","pi:");
}
```

 若要更正此警告，请传递缺少的参数，如以下代码所示：

```
#include <stdio.h>
#include <string.h>

void f()
{
  char buff [25];
  sprintf(buff,"%s %f","pi:",3.1415);
}
```

 下面的示例代码使用 `sprintf_s` 的安全字符串操作函数来更正此警告：

```
#include <stdio.h>
#include <string.h>

void f()
{
  char buff [25];
  sprintf_s( buff, 25,"%s %f", "pi:",3.1415 );
}
```
