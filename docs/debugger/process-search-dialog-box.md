# <a name="process-search-dialog-box"></a>“进程搜索”对话框
使用此对话框来查找和选择的节点中的特定进程[进程视图](../debugger/processes-view.md)。 若要显示此对话框中，将焦点移至**进程**视图窗口。 然后选择**查找进程**从**搜索**菜单。

 以下设置将可用：

 **进程**要搜索的进程 ID。

 **模块**模块字符串进行搜索。

 **向上或向下搜索方向**搜索的初始传递方向。
