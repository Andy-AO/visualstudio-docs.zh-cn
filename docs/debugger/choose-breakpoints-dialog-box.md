# <a name="choose-breakpoints-dialog-box"></a>“选择断点”对话框
尝试从“新建断点”对话框的不明确输入中设置断点时，将打开“选择断点”对话框 。 不明确输入的一个常见示例是重载函数名称。

 “断点”列出可基于不明确输入设置的可能断点。 你可以选择一个或多个要设置的断点。

 “所有”会选择“断点”框中列出的所有断点 。

 “无”将清除“断点”框中列出的所有断点 。
