# <a name="spy-menu-commands"></a>“监视”菜单命令
**Spy**菜单提供命令来使用 Spy + + Windows 和视图。

 **Windows**打开新的 Windows 视图。

 **进程**打开新的进程视图。

 **线程**打开新的线程视图。

 **日志消息**打开新的消息视图，并打开[消息选项对话框](../debugger/message-options-dialog-box.md)的视图。

 **查找窗口**显示器**FindWindow**对话框可以使用鼠标选择窗口。

 **退出**关闭 Spy + + 实用工具。
