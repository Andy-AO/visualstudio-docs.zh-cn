# <a name="tree-menu-commands"></a>“树”菜单命令
**树**菜单提供用于展开或折叠视图命令。

 **展开一级**通过显示其下的一个级别展开系统视图中当前选定的项。

 **展开分支**展开当前选定项下的所有级别。

 **全部展开**展开此项，并显示系统视图中的所有级别。

 **折叠**折叠当前选定项下的所有层。
