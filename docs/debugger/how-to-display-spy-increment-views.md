# <a name="how-to-display-spy-views"></a>如何：显示 Spy++ 视图
当 Spy + + 启动时，它将在[Windows 视图](../debugger/windows-view.md)中打开，该树视图显示计算机上的所有可用窗口和控件。 此根窗口的标题为 "Windows 1"。 还有另外三个可用的 Spy + + 视图： "[消息" 视图](../debugger/messages-view.md)、"进程"[视图](../debugger/processes-view.md)和 "线程"[视图](../debugger/threads-view.md)。 您可以从**Spy**菜单中显示所需的视图。 有关详细信息，请参阅[Spy + + 视图](../debugger/spy-increment-views.md)。
