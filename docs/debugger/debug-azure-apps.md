# <a name="debug-azure-services-in-visual-studio"></a>在 Visual Studio 中调试 Azure 服务

可以使用 Visual Studio 在不同的场景中调试 Azure 服务：

- 若要调试在 Azure 应用服务中托管的生产应用，请参阅[使用 Snapshot Debugger 调试实时 ASP.NET 应用](../debugger/debug-live-azure-applications.md)。

- 若要调试托管在 Azure 虚拟机或 Azure 虚拟机规模集中的生产应用，请参阅[使用 Snapshot Debugger 调试 azure 虚拟机和 azure 虚拟机规模集 ASP.NET](../debugger/debug-live-azure-virtual-machines.md)。

- 若要调试 Azure Kubernetes 服务中托管的生产应用，请参阅[使用 Snapshot Debugger 调试 live ASP.NET Azure Kubernetes Services](../debugger/debug-live-azure-kubernetes.md)。

- 若要使用 Application Insights 调试生产应用并将其托管在 Azure App Service 或 Service Fabric 中，请参阅[在 .net 应用中的异常上调试快照](/azure/application-insights/app-insights-snapshot-debugger)。

- 若要在 IIS （Azure App Service 或 Azure VM）上进行远程调试，请参阅[在 azure 上远程调试 ASP.NET](remote-debugging-azure.md)ASP.NET。

- 若要在 Azure Service Fabric 上远程调试 ASP.NET，请参阅[调试远程 Service Fabric 应用程序](/azure/service-fabric/service-fabric-debugging-your-application#debug-a-remote-service-fabric-application)
