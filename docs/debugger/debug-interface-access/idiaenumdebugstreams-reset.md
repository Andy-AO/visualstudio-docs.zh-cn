# <a name="idiaenumdebugstreamsreset"></a>IDiaEnumDebugStreams::Reset
将枚举序列重置到开始处。

## <a name="syntax"></a>语法

```C++
HRESULT Reset ();
```

## <a name="return-value"></a>返回值
 返回 `S_OK`。
