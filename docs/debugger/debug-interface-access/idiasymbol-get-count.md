# <a name="idiasymbolget_count"></a>IDiaSymbol::get_count
检索列表或数组中的项数。

## <a name="syntax"></a>语法

```C++
HRESULT get_count ( 
   DWORD* pRetVal
);
```

#### <a name="parameters"></a>参数
 `pRetVal`

弄返回列表或数组中的项数。

## <a name="return-value"></a>返回值
 如果成功，将返回 `S_OK`;否则，将返回 `S_FALSE` 或错误代码。

> [!NOTE]
> @No__t_0 的返回值意味着该属性对符号不可用。
