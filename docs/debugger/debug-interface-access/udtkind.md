# <a name="udtkind"></a>UdtKind
描述用户定义类型（UDT）的各种类型。

## <a name="syntax"></a>语法

```C++
enum UdtKind {
    UdtStruct,
    UdtClass,
    UdtUnion,
    UdtInterface
};
```

## <a name="elements"></a>元素
UdtStruct UDT 是一种结构。

UdtClass UDT 是一个类。

UdtUnion UDT 是一个联合。

UdtInterface UDT 是一个接口。

## <a name="remarks"></a>备注
此枚举中的值由[IDiaSymbol：： get_udtKind](../../debugger/debug-interface-access/idiasymbol-get-udtkind.md)方法返回。

## <a name="requirements"></a>要求
标头： cvconst
