# <a name="idiaenumdebugstreamdatagetname"></a>IDiaEnumDebugStreamData::get_name
检索调试数据流的名称。

## <a name="syntax"></a>语法

```C++
HRESULT get_Name ( 
   BSTR * pRetVal
)
```

#### <a name="parameters"></a>参数
 pRetVal

[out]返回的调试数据流的名称。

## <a name="return-value"></a>返回值
 如果成功，则返回`S_OK`; 否则为返回错误代码。
