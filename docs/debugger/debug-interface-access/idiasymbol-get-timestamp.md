# <a name="idiasymbolget_timestamp"></a>IDiaSymbol::get_timeStamp
检索基础可执行文件的时间戳。

## <a name="syntax"></a>语法

```C++
HRESULT get_timeStamp ( 
   DWORD* pRetVal
);
```

#### <a name="parameters"></a>参数
 `pRetVal`

弄返回基础可执行文件的时间戳。

## <a name="return-value"></a>返回值
 如果成功，将返回 `S_OK`;否则，将返回 `S_FALSE` 或错误代码。

> [!NOTE]
> @No__t_0 的返回值意味着该属性对符号不可用。
