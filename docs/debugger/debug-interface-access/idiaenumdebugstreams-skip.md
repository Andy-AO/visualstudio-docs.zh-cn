# <a name="idiaenumdebugstreamsskip"></a>IDiaEnumDebugStreams::Skip
跳过枚举序列中指定数目的调试流。

## <a name="syntax"></a>语法

```C++
HRESULT Skip ( 
   ULONG celt
);
```

#### <a name="parameters"></a>参数
 `celt`

中枚举序列中要跳过的调试流的数量。

## <a name="return-value"></a>返回值
 如果成功，将返回 `S_OK`;否则，如果没有其他要跳过的记录，将返回 `S_FALSE`。
