# <a name="idiasymbolgetbasesymbol"></a>IDiaSymbol::get_baseSymbol
检索从其基于指针的符号。

## <a name="syntax"></a>语法

```C++
HRESULT get_baseSymbol(
   IDiaSymbol** pRetVal);
```

#### <a name="parameters"></a>参数
 `pRetVal`

[out]指向从中基于指针的符号的指针。

## <a name="return-value"></a>返回值
 如果成功，则返回`S_OK`; 否则为返回`S_FALSE`或错误代码。
