# <a name="idiasegmentgetwrite"></a>IDiaSegment::get_write
检索一个标志，指示是否可以修改段。

## <a name="syntax"></a>语法

```C++
HRESULT get_write ( 
   BOOL* pRetVal
);
```

#### <a name="parameters"></a>参数
 `pRetVal`

[out]返回`TRUE`如果段可写; 否则为返回`FALSE`。

## <a name="return-value"></a>返回值
 如果成功，则返回 `S_OK`。 返回`S_FALSE`如果此属性不受支持。 否则，返回错误代码。
