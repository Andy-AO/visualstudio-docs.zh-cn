**ID：** Microsoft.VisualStudio.Workload.TestController

**描述：** 将自动测试分发到多台计算机

### <a name="components-included-by-this-workload"></a>此工作负载所包含的组件

--- | --- | --- | ---
Microsoft.VisualStudio.ComponentGroup.TestTools.TestController | 测试控制器核心功能 | 16.0.28315.86 | 必需
