## <a name="express-for-windows-desktop"></a>Express for Windows Desktop

**ID：** Microsoft.VisualStudio.Workload.WDExpress

**描述：** 通过语法感知代码编辑、源代码管理和工作项管理，生成原生和托管应用，如 WPF、WinForms 和 Win32。 现已支持 C#、Visual Basic 和 Visual C++。
