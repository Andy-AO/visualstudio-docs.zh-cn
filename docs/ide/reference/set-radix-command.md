# <a name="set-radix-command"></a>“设置基数”命令
设置或返回用来显示整数值的数值基数。

## <a name="syntax"></a>语法

```cmd
Debug.SetRadix [10 | 16 | hex | dec]
```

## <a name="arguments"></a>参数
`10` 或 `16` 或 `hex` 或 `dec`

可选。 指示十进制（10 或 dec）或十六进制（16 或 hex）。 如果省略某个参数，则返回当前基数值。

## <a name="example"></a>示例
此示例将环境设置为以十六进制格式显示整数值。

```cmd
>Debug.SetRadix hex
```
