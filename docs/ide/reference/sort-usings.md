# <a name="sort-usings"></a>对 Using 排序

此重构适用于：

- C#

**功能：** 对 Using 排序。

**使用时机：** 想要对文件顶部的 `using` 指令排序，来让它们按字母顺序排序。 

操作原因：  这样便于查找 using 指令。

## <a name="how-to"></a>操作说明

1. 从菜单栏中选择“编辑”。 
2. 选择“Intellisense” > “对 Using 排序”   。
