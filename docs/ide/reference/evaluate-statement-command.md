# <a name="evaluate-statement-command"></a>Evaluate Statement 命令

计算并显示给定的语句。

## <a name="syntax"></a>语法

```cmd
>Debug.EvaluateStatement text
```

## <a name="arguments"></a>自变量

`text`

必需。 要评估的语句。

## <a name="example"></a>示例

```cmd
>Debug.EvaluateStatement args.Length
```
