# <a name="visual-studio-ide-64-bit-support"></a>Visual Studio IDE 64 位支持

可使用 Visual Studio 将应用程序设置为面向不同平台（包括 64 位平台）。 若要深入了解 Visual Studio 中对 64 位平台的支持，请参阅 [64 位应用程序](/dotnet/framework/64-bit-apps)。
