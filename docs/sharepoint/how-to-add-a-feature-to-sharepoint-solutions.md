# <a name="how-to-add-a-feature-to-sharepoint-solutions"></a>如何：将功能添加到 SharePoint 解决方案
  创建 SharePoint 解决方案时，向解决方案添加一个功能和一个包。 可以添加更多的功能项来组织不同类型的 SharePoint 文件。

### <a name="to-add-a-feature-through-solution-explorer"></a>若要添加功能通过解决方案资源管理器

1. 在中**解决方案资源管理器**，打开快捷菜单**功能**，然后选择**添加功能**。
