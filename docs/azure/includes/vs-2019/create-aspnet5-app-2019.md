1. 在 Visual Studio“启动”窗口，选择“创建新项目”  。
1. 依次选择“ASP.NET Core Web 应用程序”和“下一步”   。
1. 为新应用程序指定名称（或使用默认名称），并选择“创建”  。
1. 选择“Web 应用程序”  。
1. 通过使用“为 HTTPS 配置”复选框，选择是否需要 SSL 支持  。
1. 勾选“启用 Docker 支持”复选框  。
1. 选择所需的容器类型（Windows 或 Linux），然后单击“创建”  。
