1. 在 Visual Studio“启动”窗口，选择“创建新项目”  。
1. 选择“ASP.NET Core Web 应用程序”，然后选择“下一步”   。
1. 输入新应用程序的名称（或使用默认名称），然后选择“创建”  。
1. 选择“Web 应用程序”  。
1. 通过选择或清除“HTTPS 配置”复选框，选择是否需要 SSL 支持  。
1. 选择“启用 Docker 支持”复选框  。
1. 选择所需的容器类型（Windows 或 Linux），然后选择“创建”  。
