# <a name="imanagedaddinunload"></a>IManagedAddin::Unload
  只在托管 VSTO 外接程序卸载之前调用。

## <a name="syntax"></a>语法

```csharp
HRESULT Unload();
```

## <a name="return-value"></a>返回值
 HRESULT 值，指示方法是否已成功完成。

## <a name="remarks"></a>备注
 当前版本的 Microsoft Office 不调用此方法。 此方法保留供将来使用。
