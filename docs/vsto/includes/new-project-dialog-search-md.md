> 还可以通过在 "**创建新项目**" 对话框顶部的 "搜索" 文本框中键入 Office 产品的名称来搜索模板。 例如，键入**excel**将显示 excel 的所有项目模板。
还可以通过在 "项目类型" 下拉框中选择 " **office** "，来筛选模板列表以仅显示 office 模板。  有关详细信息，请参阅[在 Visual Studio 中创建新项目](../../ide/create-new-project.md)。
