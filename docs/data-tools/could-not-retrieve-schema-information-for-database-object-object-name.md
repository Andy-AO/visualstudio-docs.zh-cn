# <a name="could-not-retrieve-schema-information-for-database-object-object-name"></a>未能获取数据库对象 \<对象名称> 的架构信息

将“服务器资源管理器”或“数据库资源管理器”中的对象复制到剪贴板，从数据库中删除，然后再粘贴到设计器上时，通常会显示此消息。 因为该数据库对象不再存在，所以显示此消息。
