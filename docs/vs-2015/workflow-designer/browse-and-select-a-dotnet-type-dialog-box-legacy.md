# <a name="browse-and-select-a-net-type-dialog-box-legacy"></a>“浏览并选择 .NET 类型”对话框（旧版）
本主题介绍如何使用旧 [!INCLUDE[wfd1](../includes/wfd1-md.md)] 中的 "**浏览并选择 .Net 类型**" 对话框。 在需要面向 [!INCLUDE[wfd2](../includes/wfd2-md.md)] 或 [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] 时，请使用旧 [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)]。

 在 "**属性**" 窗口中，当您选择在引用的程序集中采用 .NET Framework 类型的属性时，将在属性的文本框的末尾出现省略号 **[...]** 。 单击 " **[...]** " 将打开 "**浏览并选择 .net 类型**" 对话框。 在此对话框中，可以从引用的程序集的一个树视图中选取一个类型。 例如，在使用活动设计器时，在 "**属性**" 窗口中，单击 "**基类**" 省略号 **[...]** ，从引用的程序集树中为活动选择另一个基类。

 下表描述了 "**浏览并选择 .Net 类型**" 对话框的用户界面（UI）元素。

|UI 元素|描述|
|----------------|-----------------|
|**类型名称：**|当前选定的类型的名称。|
|**Type**|左窗格显示引用程序集的树视图。 右窗格显示在左窗格中所选引用程序集的选择的可用类型。|
