# <a name="theme-configuration-dialog-box-legacy"></a>“主题配置”对话框（旧版）
本主题介绍如何使用**主题配置**对话框中，在旧[!INCLUDE[wfd1](../includes/wfd1-md.md)]。 在需要面向 [!INCLUDE[wfd2](../includes/wfd2-md.md)] 或 [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] 时，请使用旧 [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)]。  
  
 主题定义工作流的背景和前景颜色、样式、图标及其他可视元素。 可以保存主题以供其他工作流重复使用。  
  
 创建和编辑通过使用主题**主题配置**对话框。 若要打开对话框中，选择**创建新主题**上**工作流**菜单中或右击工作流设计图面，然后选择**创建新主题**。  
  
 下表介绍的用户界面 (UI) 元素**主题配置**对话框。  
  
|UI 元素|描述|  
|----------------|-----------------|  
|**主题名称：**|标识中的主题的名称[主题、 工作流设计器选项对话框 （旧版）](../workflow-designer/themes-workflow-designer-options-dialog-box-legacy.md)。 将为新主题生成可更改的名称。|  
|**主题位置：**|主题文件的文件名和路径。 将基于已生成的主题名称为新主题生成可更改的文件名。 如果更改生成的主题名称，则可能需要更改文件名以与主题名称匹配。|  
|**...**|单击以选择位置来保存使用 .wtm 文件扩展名的工作流主题文件。 所选的路径将中看到**主题位置**文本框。|  
|**选择设计器并配置属性：**|左窗格列出了可为其自定义主题的活动的树视图。 选择树视图中的活动，该活动的主题属性将显示在树视图窗格右边的属性窗格中。 单击属性以更改其值。|  
|**预览**|单击以显示一个窗口来预览属性更改的效果。|
