# <a name="c28219"></a>C28219
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28219：批注中的参数应为枚举成员  
  
 批注的参数应为命名类型的成员 `enum` ，但遇到了其他某个符号; 请使用该类型的成员 `enum` 。 这通常表示存在错误编码的批注宏。
