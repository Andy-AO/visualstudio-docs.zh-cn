# <a name="ca0070"></a>CA0070

当代码分析分析程序集并遇到权限属性并且满足以下条件之一时，就会发生此错误：

- 代码分析找不到包含属性的程序集。

- 代码分析没有加载程序集的权限。

- 代码分析加载的程序集不包含特性。

在大多数情况下，可通过确保使用最新版本的代码分析工具来解决该问题。
