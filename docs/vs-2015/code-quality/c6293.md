# <a name="c6293"></a>C6293
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6293:For 循环定义错误： 从最小值向下计数  
  
 此警告意味着 for 循环可能不会按预期。 便会出现一个循环从最小值，但具有更高版本的终止条件。  
  
 一个已签名，或无符号 — 以及负增量索引变量将导致循环计数，直到发生溢出。 这将终止循环。  
  
## <a name="example"></a>示例  
 下面的示例代码将生成此警告：  
  
```  
void f( )  
{  
   signed char i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
  
```  
  
 若要更正此警告，请使用以下代码：  
  
```  
void f( )  
{  
   signed char i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
