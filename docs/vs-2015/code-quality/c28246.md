# <a name="c28246"></a>C28246
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28246：函数 "\<名称 >" 参数 "\<参数 >" 的批注与参数的类型不匹配  
  
 在创建批注时，__deref 运算符被应用于非指针类型。
