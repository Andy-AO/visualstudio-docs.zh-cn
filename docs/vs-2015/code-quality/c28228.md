# <a name="c28228"></a>C28228
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28228:批注参数：在批注中不能使用类型  
  
 此警告意味着参数属于不支持的类型。 批注只能使用一组有限的类型作为参数。 如果正对批注使用标准宏，则这应不是可能实现的；此警告在正常使用时应不显示。
