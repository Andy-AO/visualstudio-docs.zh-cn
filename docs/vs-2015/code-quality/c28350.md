# <a name="c28350"></a>C28350
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28350：批注 \<annotation> 描述了一种并非有条件地适用的情况。  
  
 通常，当批注应用于检查 C/C++ 类型的位置时，将会生成此警告。
