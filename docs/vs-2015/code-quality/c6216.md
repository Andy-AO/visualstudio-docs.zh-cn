# <a name="c6216"></a>C6216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6216：在语义不同的整数类型之间进行编译器插入的强制转换： Boolean 类型转换为 HRESULT  
  
 此警告意味着在没有显式强制转换的情况下，布尔值作为 `HRESULT` 使用。 布尔类型通过非零值指示成功;`HRESULT` 中的成功（`S_OK`）表示值0。 将返回布尔值 false 的函数的典型失败值作为 `HRESULT`进行测试时为成功状态。 这可能会导致不正确的结果。  
  
## <a name="example"></a>示例  
 下面的代码将生成此警告：  
  
```  
#include <windows.h>  
BOOL IsEqual(REFGUID, REFGUID);  
  
HRESULT f( REFGUID riid1, REFGUID riid2 )  
{  
  // code ...  
  return IsEqual(riid1, riid2);    
}  
```  
  
 若要更正此警告，请使用以下代码：  
  
```  
#include <windows.h>  
BOOL IsEqual(REFGUID, REFGUID);  
  
HRESULT f( REFGUID riid1, REFGUID riid2 )  
{  
  if (IsEqual(riid1, riid2) == TRUE)  
  {  
    // code ...  
    return S_OK;  
  }  
  else  
  {  
    // code ...  
    return E_FAIL;  
  }  
}  
```  
  
 对于此警告，`SCODE` 类型等效于 `HRESULT`。  
  
 有关详细信息，请参阅[SUCCEEDED 宏](https://go.microsoft.com/fwlink/?LinkId=92738)和[FAILED 宏](https://go.microsoft.com/fwlink/?LinkId=180875)。
