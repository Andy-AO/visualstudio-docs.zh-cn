# <a name="c28280"></a>C28280
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28280：对于符号，已找到 "end"，但没有匹配的 "begin"  
  
 批注语言支持开始和结束 (`{` `}` 在 C) 构造中，并且配对会导致不平衡。 如果使用了宏，则此情况可避免。
