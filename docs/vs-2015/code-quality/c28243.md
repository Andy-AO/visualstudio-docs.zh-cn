# <a name="c28243"></a>C28243
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28243:参数上函数的批注需要的取消引用次数多于已批注的实际类型所允许的次数  
  
 批注 `__deref` 运算符的数量多于由参数类型定义的指针级别数。 通过改变批注中的数量或所引用参数的指针级别，可更正此警告。
