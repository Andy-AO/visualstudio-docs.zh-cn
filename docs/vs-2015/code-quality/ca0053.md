# <a name="ca0053"></a>CA0053

无法加载规则程序集。

这表示无法通过反射加载规则程序集。 如果用户具有不正确，生成的自定义规则，可能出现此问题或分析工具不能加载的程序集出于其他原因，如文件系统访问被拒绝。
