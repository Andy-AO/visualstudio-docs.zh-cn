# <a name="c6506"></a>C6506
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6506： 无效的批注：\<名称 > 属性只能用于指针或数组类型的值  
  
 此警告意味着在不属于指针或数组类型上使用的属性。 可以在所有数据类型上使用访问、 Tainted 和有效的属性。 其他属性，如 ValidBytesConst、 ValidElementsConst、 ElementSize 和 NullTerminted 支持指针、 指向成员或数组类型。 属性和支持的数据类型的完整列表，请参阅[批注属性](https://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d)。  
  
## <a name="example"></a>示例  
 下面的代码生成此警告：  
  
```cpp  
#include<sal.h>  
void f(_Out_ char c)  
{  
    c = 'd';  
}  
```  
  
 若要更正此警告，请使用指针或数组类型，如下面的示例代码中所示：  
  
```cpp  
#include<sal.h>  
void f(_Out_ char *c)  
{  
    *c = 'd';  
}  
```
