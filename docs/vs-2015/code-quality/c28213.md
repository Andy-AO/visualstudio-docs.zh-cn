# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28213： `_Use_decl_annotations_` 必须使用批注来引用，而无需修改以前的声明。 \<why>  
  
 当引用之前的声明，但没有必要的 `_Use_decl_annotations_` 批注时，将报告此警告。
