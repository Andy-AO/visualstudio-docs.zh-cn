# <a name="suppress-warnings-by-using-the-suppressmessage-attribute"></a>使用 SuppressMessage 特性禁止显示警告
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

通常，这会很有用，这是为了让团队成员了解代码已评审并且确定应禁止显示警告，这通常很有用。 在源抑制（ISS）中，开发人员可以将禁止显示警告的属性放置在生成警告的位置附近。 可以直接将 ISS 特性添加到源文件中，也可以使用 [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] IDE 中的快捷菜单。
