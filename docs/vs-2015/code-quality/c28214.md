# <a name="c28214"></a>C28214
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28214：特性参数名称必须为 p1 .。。p9  
  
 此警告表示当构造批注声明时，参数名称限制为 p1 .。。p9. 此警告在正常使用时不应发生。
