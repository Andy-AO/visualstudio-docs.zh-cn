# <a name="c6296"></a>C6296
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6296:For 循环定义错误：循环体仅执行一次  
  
 此警告意味着 for 循环可能不会按预期。 当索引是无符号的从零开始向下计数循环时，其正文是仅运行一次。  
  
## <a name="example"></a>示例  
 下面的代码生成此警告：  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
```  
  
 若要更正此警告，请使用以下代码：  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
