# <a name="c28223"></a>C28223
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28223:未找到批注和参数所需的标记/标识符  
  
 此警告意味着批注参数应为标识符，并且遇到了一些其他符号。 这通常表示存在错误编码的批注宏。 使用 C 或C++此位置中的关键字会导致此错误。
