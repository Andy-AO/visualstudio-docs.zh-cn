# <a name="ca0505"></a>CA0505

由于定义了 superceding 属性，将忽略不推荐使用的属性。

当指定的不推荐使用和 superceding 代码分析属性包含项目或生成配置文件中的值时，将生成警告 CA505。 若要解决此问题，请删除不推荐使用的属性。
