# <a name="c28225"></a>C28225
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28225：在批注中找不到所需参数的正确数目  
  
 此警告表明命名批注使用的参数数目不正确。 如果正对批注使用标准宏，则这应不是可能实现的；此警告在正常使用时应不显示。
