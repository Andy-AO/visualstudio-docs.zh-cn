# <a name="c6504"></a>C6504
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6504：无效的批注：属性只能用在指针值、指向成员的指针或数组类型的值上  
  
 此警告意味着在不兼容的数据类型上使用属性。 有关属性支持的数据类型的详细信息，请参阅 [批注属性](https://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d)。  
  
## <a name="example"></a>示例  
 下面的代码会生成此警告，因为不能对引用数据类型使用 `_Null_` 属性。  
  
```cpp  
#include<sal.h>  
  
class Point  
{  
public:  
    //  members  
};  
  
void f(_Pre_ _Null_ Point& pt)  
{  
    // code ...  
}  
  
```  
  
 若要更正此警告，请使用以下代码：  
  
```cpp  
#include<sal.h>  
  
class Point  
{  
public:  
    //  members  
};  
  
void f(_Pre_ _Null_  Point* pt)  
{  
    // code ...  
}  
```  
  
 前面所示的缺陷代码还会生成警告 [C6516](../code-quality/c6516.md) ，因为属性冲突导致无效的批注。
