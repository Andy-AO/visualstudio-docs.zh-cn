# <a name="c6510"></a>C6510
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6510:无效的批注：只能在它的元素的缓冲区上使用 NullTerminated 属性的整数或指针类型：函数\<函数 >\<参数 >。  
  
 此警告指示错误使用**NullTerminated**属性 (以`_z`)。 只能对指针或数组类型使用此属性类型。  
  
## <a name="example"></a>示例  
 下面的代码生成此警告：  
  
```  
#include <sal.h>  
  
void f(_In_z_ char x)  
{  
    // code ...  
}  
```  
  
 若要更正此警告，请使用以下代码：  
  
```  
#include <sal.h>  
  
void f(_In_z_ char * x)  
{  
    // code ...  
}  
```
