# <a name="c28163"></a>C28163
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28163:该函数应永远不会调用从在一个 try / except 块  
  
 当函数的类型时，将报告此警告，永远不应括在`try/except`中找到该块`try/except`块。 代码分析工具至少发现了一个路径，其中，所调用的函数都位于 `try/except` 块中。
