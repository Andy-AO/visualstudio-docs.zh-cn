# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28020： \<expr> 此调用中的表达式不是 true  
  
 当 \_ \_ 列出的表达式不为 true 时，将报告此警告。 这通常表示存在错误的参数。  
  
 如果在函数声明时发生此警告，则表示批注指明了一种不可能的条件。
