# <a name="window-menu-commands"></a>“窗口”菜单命令
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

“窗口”菜单提供用于管理 Spy++ 窗口的命令。  
  
 **Cascade**  
 以级联模式重排所有窗口。  
  
 **横向平铺**  
 重新排列水平平铺模式的所有窗口。  
  
 **垂直平铺**  
 重新排列垂直平铺模式中的所有窗口。  
  
 **排列图标**  
 如果 windows 已最小化，则重新排列视图图标。  
  
 **关闭/全部关闭**  
 关闭 Spy + + 中的一个或所有窗口。  
  
 **“刷新”**  
 刷新当前所选视图。  
  
 **窗口**  
 允许选择特定窗口。
