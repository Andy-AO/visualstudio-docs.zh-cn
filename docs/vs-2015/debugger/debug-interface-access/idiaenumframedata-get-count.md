# <a name="idiaenumframedataget_count"></a>IDiaEnumFrameData::get_Count
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

检索帧数据元素的数目。  
  
## <a name="syntax"></a>语法  
  
```cpp#  
HRESULT get_Count (   
   LONG* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 pRetVal  
 弄返回帧数据元素的数目。  
  
## <a name="return-value"></a>返回值  
 如果成功， `S_OK` 则返回; 否则返回错误代码。
