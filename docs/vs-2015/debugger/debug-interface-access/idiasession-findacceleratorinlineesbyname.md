# <a name="idiasessionfindacceleratorinlineesbyname"></a>IDiaSession::findAcceleratorInlineesByName
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

返回与指定内联函数名称对应的内联帧的符号的枚举。  
  
## <a name="syntax"></a>语法  
  
```cpp#  
HRESULT findAcceleratorInlineeLinesByName (   
   LPCOLESTR             name,  
   DWORD                 option,  
   IDiaEnumSymbols**     ppResult  
);  
```  
  
#### <a name="parameters"></a>参数  
 `name`  
 中要搜索的被内联方函数名称。  
  
 `option`  
 中搜索对应于的内联框架时要使用的名称搜索选项 `name` 。 有关详细信息，请参阅 [NameSearchOptions 枚举](../../debugger/debug-interface-access/namesearchoptions.md)。  
  
 `ppResult`  
 弄指向 `IDiaEnumSymbols` 使用结果进行初始化的接口指针的指针。  
  
## <a name="return-value"></a>返回值  
 如果成功， `S_OK` 则返回; 否则返回错误代码。  
  
## <a name="remarks"></a>备注  
 此函数仅在快捷键存根函数内搜索内联函数。 它忽略本机 c + + 过程记录。
