# <a name="idiaenumsourcefilesreset"></a>IDiaEnumSourceFiles::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

将枚举序列重置到开始处。  
  
## <a name="syntax"></a>语法  
  
```cpp#  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>返回值  
 返回 `S_OK`。
