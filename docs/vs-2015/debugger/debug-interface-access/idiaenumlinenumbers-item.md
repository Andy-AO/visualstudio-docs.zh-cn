# <a name="idiaenumlinenumbersitem"></a>IDiaEnumLineNumbers::Item
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

通过索引检索行号。  
  
## <a name="syntax"></a>语法  
  
```cpp#  
HRESULT Item (   
   DWORD            index,  
   IDiaLineNumber** lineNumber  
);  
```  
  
#### <a name="parameters"></a>参数  
 索引  
 中要检索的 [IDiaLineNumber](../../debugger/debug-interface-access/idialinenumber.md) 对象的索引。 索引介于0到 `count` -1 之间，其中 `count` 由 [IDiaEnumLineNumbers：： get_Count](../../debugger/debug-interface-access/idiaenumlinenumbers-get-count.md) 方法返回。  
  
 lineNumber  
 弄返回一个 [IDiaLineNumber](../../debugger/debug-interface-access/idialinenumber.md) 对象，该对象表示所需的行号。  
  
## <a name="return-value"></a>返回值  
 如果成功， `S_OK` 则返回; 否则返回错误代码。
