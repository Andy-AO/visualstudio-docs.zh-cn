# <a name="idiaenumdebugstreamdataskip"></a>IDiaEnumDebugStreamData::Skip
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

跳过枚举序列中指定数目的记录。  
  
## <a name="syntax"></a>语法  
  
```cpp#  
HRESULT Skip (   
   ULONG celt  
);  
```  
  
#### <a name="parameters"></a>参数  
 celt  
 中要在枚举序列中跳过的记录数。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回 `S_OK` ; 否则， `S_FALSE` 如果没有更多要跳过的记录，则返回。
