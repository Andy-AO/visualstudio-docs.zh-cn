# <a name="tree-menu-commands"></a>“树”菜单命令
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

“树”菜单提供用于展开或折叠视图的命令。  
  
 **展开一级**  
 在系统视图中，通过在其下显示一个级别来展开当前选定项。  
  
 展开分支  
 展开当前选定项下的所有级别。  
  
 **全部展开**  
 展开并显示系统视图中的所有级别。  
  
 **折叠**  
 折叠当前选定项下的所有级别。
