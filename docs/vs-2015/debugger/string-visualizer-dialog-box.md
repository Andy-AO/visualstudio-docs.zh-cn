# <a name="string-visualizer-dialog-box"></a>字符串可视化工具对话框
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

此对话框是 Visual Studio 调试器中包含的标准可视化工具之一。 除普通文本外，字符串可视化工具还可以计算 HTML 和 XML 字符串，这使得您能够以有意义的格式查看结果。
