# <a name="messages-menu-commands"></a>“消息”菜单命令
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**消息**菜单提供用于控制消息视图命令。  
  
 **启动/停止日志记录**  
 启动或停止消息视图。  
  
 **日志记录选项**  
 此时将打开**消息选项**对话框中，您可以在其中指定消息搜索条件。  
  
 **清除日志**  
 清除活动的消息视图的内容。
