# <a name="help-menu-commands"></a>“帮助”菜单命令
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

“帮助”菜单提供用于打开帮助文件的命令。  
  
 **Index**  
 为 Spy + + 帮助打开内容屏幕。  
  
 **使用帮助**  
 打开说明如何使用帮助的帮助文件。  
  
 **关于 Spy++**  
 显示 Spy + + 的软件版本和版权信息。
