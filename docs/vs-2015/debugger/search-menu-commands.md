# <a name="search-menu-commands"></a>“搜索”菜单命令
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

“搜索”菜单提供用于搜索特定窗口的命令。  
  
 **查找窗口/进程/线程/消息**  
 打开 "**窗口** / **进程** / **线程** / **MessageSearch** " 对话框，您可以在其中使用**FinderTool**或指定搜索条件。  
  
 **查找下一个**  
 在当前视图中搜索下一个匹配的窗口、进程、线程或消息。 仅当存在并非唯一的有效搜索结果时，此菜单项（以及相关的工具栏按钮）才可用。  
  
 **查找上一个**  
 在当前视图中搜索上一个匹配的窗口、进程、线程或消息。 仅当存在并非唯一的有效搜索结果时，此菜单项（以及相关的工具栏按钮）才可用。
