# <a name="spy-menu-commands"></a>“监视”菜单命令
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

“监视”菜单提供了用于使用 Spy++ 窗口和视图的命令。  
  
 **Windows**  
 打开一个新的窗口视图。  
  
 **进程**  
 打开新的 "进程" 视图。  
  
 **线程**  
 打开新的 "线程" 视图。  
  
 **日志消息**  
 打开新的消息视图，并打开该视图的 " [消息选项" 对话框](../debugger/message-options-dialog-box.md) 。  
  
 **查找窗口**  
 显示 " **FindWindow** " 对话框以使用鼠标选择窗口。  
  
 **退出**  
 关闭 Spy + + 实用工具。
