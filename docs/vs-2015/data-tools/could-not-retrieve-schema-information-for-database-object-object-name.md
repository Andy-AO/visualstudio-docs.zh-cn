# <a name="could-not-retrieve-schema-information-for-database-object-ltobject-namegt"></a>无法检索数据库对象的架构信息对象 &lt; 名称&gt;
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

当 **服务器资源管理器/数据库资源管理器** 中的对象复制到剪贴板，从数据库中删除，然后粘贴到设计器中时，通常会出现此消息。 因为该数据库对象不再存在，所以显示此消息。

### <a name="to-correct-this-error"></a>更正此错误

- 单击“确定”关闭对话框****。
