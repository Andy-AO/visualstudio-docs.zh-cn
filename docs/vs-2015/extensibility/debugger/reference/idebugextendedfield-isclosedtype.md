# <a name="idebugextendedfieldisclosedtype"></a>IDebugExtendedField::IsClosedType
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

确定字段是否表示闭合类型。  
  
## <a name="syntax"></a>语法  
  
```cpp#  
HRESULT IsClosedType(  
   void  
);  
```  
  
```csharp  
int IsClosedType();  
```  
  
## <a name="return-value"></a>返回值  
 如果该字段为关闭类型，则返回 `S_OK` ; 否则返回 `S_FALSE` 。
