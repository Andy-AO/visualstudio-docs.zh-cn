# <a name="modules"></a>模块
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

就调试器体系结构而言， **模块**如下：  
  
- 是一个物理代码容器，如可执行文件或 DLL。  
  
- 可以重新加载其符号并对其进行描述。 模块说明显示在 IDE 的 "模块" 窗口中。  
  
- 由调试引擎创建的 [IDebugModule2](../../extensibility/debugger/reference/idebugmodule2.md) 接口表示，用于说明模块。
