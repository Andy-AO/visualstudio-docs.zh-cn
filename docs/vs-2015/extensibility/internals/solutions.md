# <a name="solutions"></a>解决方案
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

一种解决方案是协同工作以创建应用程序的一个或多个项目的分组。 适用于解决方案的项目和状态信息存储在两个解决方案文件、 基于文本的解决方案 (.sln) 文件和二进制解决方案用户选项 (.suo) 文件。
