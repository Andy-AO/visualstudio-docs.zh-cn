# <a name="solutions"></a>解决方案
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

解决方案是一个或多个项目的分组，它们一起工作以创建应用程序。 与解决方案相关的项目和状态信息存储在两个解决方案文件中，一种基于文本的解决方案 ( .sln) 文件，而二进制解决方案用户选项 ( .suo) 文件中。
