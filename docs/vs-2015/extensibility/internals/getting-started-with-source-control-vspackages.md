# <a name="getting-started-with-source-control-vspackages"></a>源代码管理 VSPackage 入门
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

源代码管理 VSPackage 是源代码管理插件的更高级替代方法。 (请参阅[创建源代码管理插件](../../extensibility/internals/creating-a-source-control-plug-in.md)对于源的详细信息来控制插件。)源代码管理 VSPackage 启用对源控件模型、 功能和用户界面 (UI) 的完全控制和集成到[!INCLUDE[vsprvs](../../includes/vsprvs-md.md)]作为 VSPackage 的环境。
