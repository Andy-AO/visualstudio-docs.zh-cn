[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

 选择中的对象时 [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] ，该对象的公共属性将显示在 " **属性** " 窗口中。 有关详细信息，请参阅 [属性窗口概述](../extensibility/internals/properties-window-overview.md)。
