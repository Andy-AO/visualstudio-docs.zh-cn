# <a name="extending-connected-services"></a>扩展连接的服务
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio 连接的服务 SDK 允许服务提供程序创建 Visual Studio 扩展，应用开发人员可以轻松地将其添加到项目中，而无需离开 Visual Studio IDE。
