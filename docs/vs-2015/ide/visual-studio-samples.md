# <a name="visual-studio-samples"></a>Visual Studio Samples
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

可以在 [MSDN 代码库](https://code.msdn.microsoft.com/) 网站上找到 Visual Studio 示例。
