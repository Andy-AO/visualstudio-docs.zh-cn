# <a name="build-cross-platform-mobile-apps-with-visual-c"></a>通过视觉对象构建跨平台的移动应用C++
可以通过使用适用于跨平台移动开发的 Visual C++ 构建用于 Android、iOS 和 Windows 设备的跨平台代码。 这是在 Visual Studio 2015 中可用的一项可选功能，Visual Studio 2015 通过使用 Visual C++ 实现适用于 iOS、Android 和 Windows 的跨平台代码开发。  
  
 有关安装说明和代码示例，请[参阅C++适用于跨平台移动开发的视觉对象](../cross-platform/visual-cpp-for-cross-platform-mobile-development.md)。
