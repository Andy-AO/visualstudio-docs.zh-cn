# <a name="the-precision-is-out-of-range"></a>精度超出范围
您尝试将无效参数传递给**toPrecision**。 **ToPrecision**的参数必须介于1和21（含）之间。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 请确保 `toPrecision` 的参数不太大或太小。
