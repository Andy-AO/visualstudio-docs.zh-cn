# <a name="expected-"></a>预期\@
尝试创建一个变量以与使用的条件编译语句一起使用`@set`语句，但是没有按照 at 符号"**@**"的变量名称之前。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 添加 at 符号"**@**"变量名称之前。 例如：  
  
    ```JavaScript  
    @set @myvar = 1  
    ```
