# <a name="undefined-identifier"></a>未定义标识符
你尝试使用标识符的[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]编译器无法识别。 每当你使用，则返回未定义的值：  
  
- 变量不存在，  
  
- 已声明但从未遇到过的变量分配给它的值  
  
- 对象属性不存在。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 与变量声明**var**语句 (如`var`x;)。
