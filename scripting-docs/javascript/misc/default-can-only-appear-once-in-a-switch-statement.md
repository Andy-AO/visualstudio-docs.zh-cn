# <a name="default-can-only-appear-once-in-a-switch-statement"></a>“default”在“switch”语句中只能出现一次
您尝试在 switch 语句中多次使用**default**语句。 默认情况下，switch 语句中的最后一个 case 语句（这是贯穿大小写）。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 从语句中删除任何多余的**默认**事例语句 `switch` （在 switch 语句中最多使用一个默认事例语句）。
