# <a name="unexpected-quantifier-javascript"></a>意外的限定符 (JavaScript)
编写正则表达式搜索模式，创建的 pattern 元素非法的重复因子。 例如，模式  
  
```js
/^+/  
```  
  
 是非法的因为元素 ^ （输入的开头） 不能具有重复因子。 下表列出了这些元素不能具有重复因素。  
  
|元素|描述|  
|-------------|-----------------|  
|^|输入起始处|  
|$|输入结束|  
|\b|字边界|  
|\B|非字边界|  
|*|零个或多个重复项|  
|+|一个或多个重复项|  
|?|零个或一个重复项|  
|{n}|n 重复|  
|{n,}|n 个或多个重复项|  
|{n,m}|从 n 到 m 次重复，非独占|  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 请确保您的搜索模式元素包含仅法律重复因素。
