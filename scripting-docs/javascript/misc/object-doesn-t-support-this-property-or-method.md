# <a name="object-doesnt-support-this-property-or-method"></a>对象不支持此属性或方法
你指定的属性或方法不在此对象中。 并非所有对象都支持所有的属性和方法。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 检查属性和方法以确保不存在任何键入错误。  
  
- 验证对象的运行时类型是预期的对象类型。  
  
- 有关更多信息，请参见对象的文档。
