# <a name="number-expected"></a>缺少数字
试图对不是**number**的类型的对象调用**valueOf**方法，则尝试**调用此方法**。 此类调用的对象必须是**Number**类型。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 仅对**number**类型的对象调用 ValueOf 或**number** .**方法。**
