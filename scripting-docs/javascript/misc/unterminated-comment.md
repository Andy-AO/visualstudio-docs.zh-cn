# <a name="unterminated-comment"></a>未终止的注释
开始的多行注释块，但未正确终止它。 多行注释开头"/ *"的组合，并最终以反"\*/"组合。 下面是一个示例：  
  
```JavaScript  
/* This is a comment  
This is another part of the same comment.*/  
```  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 一定要终止多行注释以"* /"。
