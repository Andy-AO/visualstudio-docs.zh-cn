# <a name="expected-"></a>应有“{”
你未包含标记的函数体开头的左大括号。 组成函数的正文，即使它是单个行的代码必须包含在左和右大括号。 请注意，在函数体中的大括号的使用比在循环中更严格。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 添加标记的函数体开头的左大括号。
