# <a name="expected-catch"></a>应有“catch”
使用异常处理**尝试**阻止，但没有编写关联**捕获**语句。 异常处理机制要求可能会失败，以及在发生异常时，不应执行的代码的代码将被封装在**尝试**块。 内引发的异常**尝试**阻止使用**引发**语句，并捕获外部**重**块包含一个或多个**捕获**语句。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 添加关联**捕获**块。  
  
- 请尝试使用**最后**而不是阻止**捕获**块。
