# <a name="string-expected"></a>缺少字符串
试图对 `String` 之外的类型的对象调用**valueOf** **方法，而**不是调用方法。 此类调用的对象必须是 `String` 类型。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 仅对 `String`**类型的对象**调用 valueOf 或的方法。
