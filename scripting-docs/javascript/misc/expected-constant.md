# <a name="expected-constant"></a>应为常量
您尝试在条件编译测试语句中使用（非条件编译）变量。 条件编译测试语句的计算结果必须为常量。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 将变量替换为文本。  
  
- 将变量替换为条件编译变量。
