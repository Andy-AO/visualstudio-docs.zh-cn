# <a name="expected-"></a>应用“;”
您试图将多个语句放在一行，但不是未用分号分隔它们。 使用分号终止语句。 虽然可以将多个语句放在单个行上，必须用分号分隔每个。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 标记每个语句提供以分号的结尾。  
  
- 请确保函数调用正确使用括号。  
  
- 请确保不是否忘记了的标头中分号**为**循环。
