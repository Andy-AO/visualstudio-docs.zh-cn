# <a name="syntax-error-javascript"></a>语法错误 (JavaScript)
你创建的语句违反了 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 的一个或多个语法规则。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 在指示的行号上仔细检查程序的语法。  
  
- 查找定向的圆括号或大括号。
