# <a name="expected-while"></a>应有“while”
未在 `do...while` 循环中包含 `while` 条件。 **Do**语句必须在代码块的末尾有相应的 `while` 测试。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 在右大括号后包含 `while` 测试语句。
