# <a name="boolean-expected"></a>缺少布尔值
您尝试对类型之外的对象调用**valueOf**方法，而不是**调用的方法** `Boolean` 。 此类调用的对象必须是类型 `Boolean` 。 例如：

```JavaScript
var o = new Object;
o.f = Boolean.prototype.toString;
o.f();
```

## <a name="to-correct-this-error"></a>更正此错误

- 仅对**布尔**类型的对象调用**valueOf** **方法或布尔**方法。
