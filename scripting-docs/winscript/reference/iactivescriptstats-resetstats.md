# <a name="iactivescriptstatsresetstats"></a>IActiveScriptStats::ResetStats
重置此脚本的统计信息。  
  
## <a name="syntax"></a>语法  
  
```cpp
HRESULT ResetStats();  
```  
  
#### <a name="parameters"></a>参数  
 此方法需要任何参数。  
  
## <a name="return-value"></a>返回值  
 该方法返回 `HRESULT`。 可能的值包括（但并不限于）下表中的项。  
  
|“值”|描述|  
|-----------|-----------------|  
|`S_OK`|方法成功。|  
  
## <a name="remarks"></a>备注  
 此方法将重置此脚本的统计信息。
