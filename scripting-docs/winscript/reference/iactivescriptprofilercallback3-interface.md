# <a name="iactivescriptprofilercallback3-interface"></a>IActiveScriptProfilerCallback3 接口
提供一个方法，用于通知探查器分析的执行是否在 web 工作进程的上下文中发生。  
  
## <a name="syntax"></a>语法  
  
```cpp
interface IActiveScriptProfilerCallback3 : IActiveScriptProfilerCallback2  
```  
  
## <a name="methods"></a>方法  
  
|方法|说明|  
|------------|-----------------|  
|[IActiveScriptProfilerCallback3::SetWebWorkerId 方法](../../winscript/reference/iactivescriptprofilercallback3-setwebworkerid-method.md)|通知探查器要用于此分析会话的辅助 ID。|
