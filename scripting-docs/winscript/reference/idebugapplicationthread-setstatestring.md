# <a name="idebugapplicationthreadsetstatestring"></a>IDebugApplicationThread::SetStateString
设置线程状态的说明。  
  
## <a name="syntax"></a>语法  
  
```cpp
HRESULT SetStateString(  
   LPCOLESTR  pstrState  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pstrState`  
 [in]线程状态的说明。  
  
## <a name="return-value"></a>返回值  
 该方法返回 `HRESULT`。 可能的值包括（但并不限于）下表中的项。  
  
|“值”|描述|  
|-----------|-----------------|  
|`S_OK`|方法成功。|  
  
## <a name="remarks"></a>备注  
 此方法设置线程状态的说明。
