# <a name="idebugexpressionqueryiscomplete"></a>IDebugExpression::QueryIsComplete
确定操作已完成。  
  
## <a name="syntax"></a>语法  
  
```cpp
HRESULT QueryIsComplete();  
```  
  
#### <a name="parameters"></a>参数  
 此方法需要任何参数。  
  
## <a name="return-value"></a>返回值  
 该方法返回 `HRESULT`。 可能的值包括（但并不限于）下表中的项。  
  
|“值”|描述|  
|-----------|-----------------|  
|`S_OK`|该方法成功，并操作已完成。|  
|`S_FALSE`|该操作仍是挂起。|  
  
## <a name="remarks"></a>备注  
 此方法确定该操作已完成。
