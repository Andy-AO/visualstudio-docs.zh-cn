# <a name="ijsdebugbreakpointdelete-method"></a>IJsDebugBreakPoint::Delete 方法
删除断点。  
  
## <a name="syntax"></a>语法  
  
```cpp
HRESULT Delete(void);  
```  
  
## <a name="return-value"></a>返回值  
  
## <a name="remarks"></a>备注  
 如果对已删除的断点调用，则返回 S_FALSE。  
  
## <a name="requirements"></a>要求  
 **标头：** jscript9diag
