# <a name="ienumdebugextendedpropertyinforeset"></a>IEnumDebugExtendedPropertyInfo::Reset
将枚举序列重置到开始处。  
  
## <a name="syntax"></a>语法  
  
```cpp
HRESULT Reset (  
void  
);  
```  
  
## <a name="return-value"></a>返回值  
 返回一个有效 `HRESULT`，通常 `S_OK`。
