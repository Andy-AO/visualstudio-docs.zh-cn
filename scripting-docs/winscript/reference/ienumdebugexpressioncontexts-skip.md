# <a name="ienumdebugexpressioncontextsskip"></a>IEnumDebugExpressionContexts::Skip
跳过枚举序列中指定数量的段。  
  
## <a name="syntax"></a>语法  
  
```cpp
HRESULT Skip(  
   ULONG  celt  
);  
```  
  
#### <a name="parameters"></a>参数  
 `celt`  
 中枚举序列中要跳过的段数。  
  
## <a name="return-value"></a>返回值  
 该方法返回 `HRESULT`。 可能的值包括（但并不限于）下表中的项。  
  
|“值”|描述|  
|-----------|-----------------|  
|`S_OK`|方法成功。|  
  
## <a name="remarks"></a>备注  
 此方法跳过枚举序列中指定数量的段。
