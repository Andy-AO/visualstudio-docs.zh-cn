# <a name="idebugpropertysetvalueasstring"></a>IDebugProperty::SetValueAsString
从给定字符串中设置属性的值。  
  
## <a name="syntax"></a>语法  
  
```cpp
HRESULT SetValueAsString (  
   LPCOLESTR pszValue,  
   UINTnRadix,  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pszValue`  
 [in]要设置的值。  
  
 `nRadix`  
 [in]用于解释任何数字信息的基数。  
  
## <a name="return-value"></a>返回值  
 返回一个有效`HRESULT`，通常`S_OK`。
