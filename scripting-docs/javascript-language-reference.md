# <a name="javascript-language-reference"></a>JavaScript 语言参考

JavaScript 是一种可嵌入网页和其他应用程序的脚本语言。  
  
JavaScript 的 Microsoft 实现符合 ECMAScript 语言规范第5版。 它还提供了 ECMA 标准中未包含的附加功能。

> [!IMPORTANT]
> Microsoft 通过将 Microsoft 的 JavaScript API[引用的所有](https://docs.microsoft.com)500 多个页面重定向到其 MDN 对应项，加入了社区范围内的 premiere 开发参考，并迁移了我们的旧式 JS 扩展参考，[以便进行历史](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference)和维护。
