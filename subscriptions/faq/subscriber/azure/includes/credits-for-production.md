## <a name="can-i-use-my-monthly-azure-credits-to-run-production-applications"></a>我能否使用每月 Azure 额度来运行生产应用程序？

Visual Studio 订阅者的每月 Azure 额度仅用于单独的开发/测试用途。 要运行生产工作负载，需升级到付费的 Azure 订阅，例如[即用即付](https://azure.microsoft.com/offers/ms-azr-0003p/)定价。
